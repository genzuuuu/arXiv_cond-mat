### 核心主题：具身智能与机器人学习

#### 1. 灵巧手与操作 (Dexterous Hand & Manipulation)
*   **Belt-Finger: An Affordable Soft Belt-Driven Gripper for Dexterous In-Hand Manipulation** ([46](https://arxiv.org/abs/2606.20193)): 提出一种基于软带驱动的平行夹爪升级模块，增加三个自由度（平移、俯仰、滚转），实现低成本的手内灵巧操作。集成了模型预测控制和轻量级遥操作界面。
*   **From Construction to Injection: Edit-Based Fingerprints for Large Language Models** ([65](https://arxiv.org/abs/2606.20549)): 提出一个数据驱动的框架，通过学习人体演示数据（超过400万帧）直接生成机器人手的设计。利用逆运动学匹配指尖位置进行优化，并制造出可实现零样本抓取的6-DoF通用手和任务特定的低DoF手。
*   **PTLD: Sim-to-real Privileged Tactile Latent Distillation for Dexterous Manipulation** ([86](https://arxiv.org/abs/2603.04531)): 提出一种新的sim-to-real方法，不需要模拟触觉传感器。通过在真实世界收集带有特权信息的触觉策略数据，蒸馏出对触觉输入鲁棒的状态估计器，显著提升了灵巧手内操作（如旋转物体）的性能。
*   **Generating Robot Hands from Human Demonstrations** ([65](https://arxiv.org/abs/2606.20549)): 一个数据驱动的框架，通过学习人体演示数据直接生成优化的机器人手设计。通过逆运动学匹配指尖位置，结合强化学习加速设计搜索，制造出的手在遥操作中展现出高精度。
*   **Transferring Contact, Not Just Motion: Compliant Grasping Across Dexterous Hands** ([95](https://arxiv.org/abs/2606.15516)): 提出一种跨异构灵巧手的力-位接口，将运动意图和接触力统一表征。通过系统辨识校准力矩为物理单位，实现了可迁移的柔顺抓取策略，使得在一个手上学习的行为可以直接在另一个不同结构的手上使用。
*   **VibeCheck: Using Active Acoustic Tactile Sensing for Contact-Rich Manipulation** ([72](https://arxiv.org/abs/2504.15535)): 构建了一个主动声学传感夹爪，通过发送和接收声学振动来感知物体的属性（材质、抓取位置、内部结构）和接触状态。并将该传感器用于插销任务，学习出鲁棒的模仿学习策略。
*   **TaCauchy: An Extensible FEM Framework for Vision-Based Tactile Simulation** ([59](https://arxiv.org/abs/2606.20426)): 提出TaCauchy，一个基于有限元方法的视觉触觉仿真框架，用于Isaac Sim。该框架可直接计算柯西应力张量，提供物理上精确的力场监督，并与多种传感器（如GelSight Mini, DIGIT）兼容，支持强化学习训练。

#### 2. 抓取与操作任务 (Grasping & Manipulation Tasks)
*   **Human Universal Grasping** ([187](https://arxiv.org/abs/2606.17054)): 提出HUG，一个利用人体演示数据学习生成多样化抓取姿态的流匹配模型。大型数据集1M-HUGs包含6,707个物体实例。模型输出可重定向到不同机器人手，实现零样本抓取，在真实世界测试中超越基线。
*   **Scaling Self-Play for End-to-End Driving** ([16](https://arxiv.org/abs/2606.19641)): 提出Gigapixel仿真器，通过渲染简化边界框世界实现高效率的自对弈，用于训练端到端自动驾驶模型。结合自对弈DAgger训练方法，使策略能够从像素观测中大规模学习，并可以迁移到现实世界数据。
*   **See-and-Reach: Precise Vision-Language Navigation for UAVs within the Field of View** ([281](https://arxiv.org/abs/2606.20045)): 针对无人机视觉语言导航，提出一个新的子任务UAV-VLN-FOV，专注于目标进入视野后的精准抵达。提出了3DG-VLN框架，通过动态3D方向线索和视觉提示进行航点预测，显著提高了到达精度。
*   **Learning to Prompt: Improving Student Engagement with Adaptive LLM-based High-School Tutoring** ([160](https://arxiv.org/abs/2606.20138)): 提出一种基于LLM的个性化辅导系统，利用14个教学特征训练提示路由模型，实现自适应的教学策略切换。A/B测试表明，该方法提高了教学效率和互动质量。
*   **Fail-RAG : A Retrieval Augmented Generation Informed Framework for Robot Failure Identification** ([13](https://arxiv.org/abs/2606.19598)): 提出Fail-RAG框架，利用检索增强生成和视觉语言模型，通过将故障图像与故障数据库进行相似性匹配，实现了对机器人操作（如拾取、放置）中各类故障的高效识别，提升准确率25个百分点。
*   **One-to-Two Acting: A Novel Framework for Single-arm Agent Action Expansion to Dual Arms** ([32](https://arxiv.org/abs/2606.19897)): 提出ExS2D框架，通过层级动作扩展，仅使用单臂演示即可实现双臂协作操作。通过分解子任务、动作映射和优先级调度，减少54.4%的执行步数，并保证零碰撞。
*   **WorkBenchMark: A LEGO-Based Assembly Benchmark with an Assembly-by-Disassembly Baseline** ([2](https://arxiv.org/abs/2606.19358)): 推出基于乐高的机器人装配基准WorkBenchMark，包含400个任务和四个复杂度等级。提出“通过拆卸进行装配”的基线方法，在规划流程上优于现代视觉-语言-动作模型。
*   **Robust Assembly State Reasoning from Action Recognition for Human-Robot Collaboration** ([45](https://arxiv.org/abs/2606.20150)): 系统研究了在人机协作中，如何从动作识别结果中准确推理装配状态。比较了逻辑、隐马尔可夫模型和神经网络等方法，发现不同方法在不同任务和失败情景下表现各异。

#### 3. 视觉-运动策略 (Visuomotor Policy)
*   **One Demo is Worth a Thousand Trajectories: Action-View Augmentation for Visuomotor Policies** ([11](https://arxiv.org/abs/2606.19586)): 提出一种有效的数据增强框架，从单次真实世界演示中生成视觉上真实的鱼眼图像序列和对应的物理可行动作轨迹。通过高斯泼溅和轨迹优化，生成带有新障碍物的场景数据，提升策略在面对分布外观测时的鲁棒性。
*   **EquiVLA: A General Framework for Rotationally Equivariant Vision-Language-Action Models** ([28](https://arxiv.org/abs/2606.19784)): 提出EquiVLA，首个具有旋转等变性的VLA模型框架。通过等变感知器和等变扩散Transformer动作头，在保持泛化能力的同时，显著减少在多种任务基准上的数据需求。
*   **MirrorDuo: Reflection-Consistent Visuomotor Learning from Mirrored Demonstration Pairs** ([40](https://arxiv.org/abs/2606.20048)): 提出MirrorDuo，一种基于反射的公式，可自动为每个演示生成镜像对应物，实现“采集一个，免费获得一个”。无论是作为数据增强策略还是结构先验，都能在相同数据预算下显著提升表现。
*   **Co-policy: Responsive Human-Robot Co-Creation for Musical Performances** ([33](https://arxiv.org/abs/2606.19914)): 提出Co-policy框架，用于人机音乐合创。将语义意图、音乐变化和视觉运动执行解耦。通过视觉运动策略（GMP）实现低延迟执行，生成互补的、物理可行的音乐表演。
*   **Start Right, Arrive Right: Asynchronous Execution via Initial Noise Selection** ([27](https://arxiv.org/abs/2606.19774)): 针对扩散策略异步执行中动作块边界不一致的问题，提出PAINT方法。通过在选择生成前的初始噪声，而非事后修正轨迹，来保证边界一致性，无需重训练或修改策略，显著提升任务性能。
*   **RoboSSM: Scalable In-context Imitation Learning via State-Space Models** ([79](https://arxiv.org/abs/2509.19658)): 将状态空间模型（SSM）应用于上下文模仿学习（ICIL），提出RoboSSM。与基于Transformer的方法相比，SSM在长上下文提示下具有线性推理复杂度和更优的外推能力，能更好地泛化到未见过的长程任务。
*   **VFILC: Accurate Frequency Extrapolations in Imitation Learning via Sampling Frequency ILC** ([41](https://arxiv.org/abs/2606.20056)): 提出VFILC方法，结合变频率模仿学习和迭代学习控制，使机器人能够精确外推学习到的运动速度到未训练的频率范围，显著减小了接触丰富任务中的频率误差。
*   **Pose6DAug: Physically Plausible Multi-view Object Swapping for Robot Data Augmentation** ([42](https://arxiv.org/abs/2606.20118)): 提出Pose6DAug，一种失败驱动的数据增强框架。通过提取成功演示中的动作轨迹和3D物体姿态，将目标物体“3D替换”为新物体，从而生成针对失败模式的新演示数据，无需额外人工采集。

#### 4. 遥操作 (Teleoperation)
*   **CoLI: A Reproducible Platform for Continuum Robot Learning via Monolithic 3D Printing and Isomorphic Teleoperation** ([56](https://arxiv.org/abs/2606.20389)): 介绍了一个开源连续体机器人平台，通过多材料3D打印实现一体成型，并采用同构遥操作界面构建了直接的逐关节控制映射，避免了运动学建模，支持模仿学习。
*   **TASC: Task-Aware Shared Control for Relational Telemanipulation** ([77](https://arxiv.org/abs/2509.10416)): 提出TASC框架，用于关系型遥操作。通过从视觉输入构建开放词汇交互图来推断用户的任务级意图，并基于此提供抓取和操作辅助，提高了任务效率和零样本泛化能力。
*   **Humanoid Everyday: A Comprehensive Robotic Dataset for Open-World Humanoid Manipulation** ([80](https://arxiv.org/abs/2510.08807)): 发布了一个大规模、多样化的仿人机器人操作数据集，包含10k条轨迹和300万帧数据。数据通过高效的遥操作流水线收集，涵盖多种任务。同时推出云端评估平台，便于标准化比较。
*   **Learning Category-level Last-meter Navigation from RGB Demonstrations of a Single-instance** ([81](https://arxiv.org/abs/2512.11173)): 提出一种以物体为中心的模仿学习框架，实现“最后一米”精准导航。基于单物体实例的RGB演示，策略通过语言驱动的分割和空间评分矩阵解码器，泛化到未见过的物体实例和复杂环境。

#### 5. Sim-to-Real
*   **PTLD: Sim-to-real Privileged Tactile Latent Distillation for Dexterous Manipulation** ([86](https://arxiv.org/abs/2603.04531)): 核心贡献是提出一种无需模拟触觉即可迁移灵巧操作策略的sim-to-real方法。详情见上文“灵巧手与操作”部分。
*   **Domain Adaptive Diffusion Policy** ([107](https://arxiv.org/abs/2602.04037)): 提出DADP，一种域自适应扩散策略框架。通过非监督解耦技术，将动态预测中的领域特定信息分离出来，并直接注入到扩散生成过程中，实现鲁棒的零样本适应，在多种运动学和操作任务中表现优异。

#### 6. 具身智能与Agent系统
*   **Playful Agentic Robot Learning** ([5](https://arxiv.org/abs/2606.19419)): 引入“玩耍式具身智能体学习”，其中编码智能体（RATs）通过自我引导的“玩耍”阶段自主探索并学习技能。这些技能被存入代码技能库，可在后续下游任务中重用，显著提升任务完成率。
*   **ENPIRE: Agentic Robot Policy Self-Improvement in the Real World** ([153](https://arxiv.org/abs/2606.19980)): 提出ENPIRE框架，使编码智能体能够在真实世界中进行闭环策略自我改进。通过自动重置、验证、策略演进和并行化的物理反馈循环，实现了无需人工干预的自主策略提升，成功完成高难度灵巧操作任务。
*   **MemoryWAM: Efficient World Action Modeling with Persistent Memory** ([66](https://arxiv.org/abs/2606.20562)): 提出MemoryWAM，一种结合高效持久记忆的世界动作模型。通过混合记忆（近期帧、事件边界帧、紧凑的gist令牌）和相关注意力机制，实现了在非马尔可夫环境中的长期记忆依赖决策，性能和效率均优于基线。
*   **DiffusionVS: A Generative Framework for Robust Visual Servoing Based on Diffusion Policy** ([4](https://arxiv.org/abs/2606.19397)): 提出一种基于扩散策略的新型视觉伺服方法，通过预测动作序列和时间一致性来提高鲁棒性，并采用在线训练策略增强泛化能力。仿真和实物实验均达到高成功率，并能作为通用模块改进现有系统。
