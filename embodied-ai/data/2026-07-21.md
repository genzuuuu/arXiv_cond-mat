### Embodied Task Planning & Long-Horizon Manipulation
*   **[Foresight Residual RL for Long-Horizon Robot Manipulation with Vision-Language-Action Models](https://arxiv.org/abs/2607.16506)**: Addresses the failure of VLA policies in long-horizon, contact-rich assembly tasks due to poor "handoff quality" between skills. It proposes a Foresight Residual RL framework that uses an offline-estimated "foresight value" to shape the final state of each sub-task, improving long-horizon success from 54.5% to 85.6% in a nut-tightening task.
*   **[Compositional Diffusion with Guided Search for Long-Horizon Planning](https://arxiv.org/abs/2601.00126)**: Proposes Compositional Diffusion with Guided Search (CDGS) to solve the "mode averaging" problem in long-horizon planning with generative models by embedding search within the diffusion denoising process, achieving oracle-level performance on seven robot manipulation tasks.
*   **[UniETP: Unifying Environments for Generalizable Embodied Task Planning](https://arxiv.org/abs/2607.18062)**: Proposes a unified interface (UniETP) integrating four simulators (AI2-THOR, VirtualHome, Habitat, BEHAVIOR) with a consistent observation/action space and an automatic dataset generation pipeline for embodied task planning.
*   **[PRISM: Multimodal Terrain Mapping for Rover Navigation in Unstructured Environments](https://arxiv.org/abs/2607.16366)**: Presents PRISM, a multimodal (RGB, depth, thermal) perception system that generates traversability maps for autonomous rover navigation in unstructured terrain using a novel Vision Transformer-based network (OmniUnet).
*   **[RoboHarness: Memory-Driven Orchestration of Heterogeneous Robot Policies for Long-Horizon Planning](https://arxiv.org/abs/2607.18060)**: Introduces RoboHarness, a framework that orchestrates heterogeneous policies (VLAs, RL, TAMP) for long-horizon tasks by using multi-modal execution memory and a Memory Bridge to characterize policy capability boundaries and stabilize handoffs.

### Dexterous Manipulation & Grasping
*   **[Predicting Grasping Compliance in Robotic Hands through Analytical-Model-Informed Neural Networks](https://arxiv.org/abs/2607.17541)**: Develops a hybrid predictive model (AMINN) that combines an analytical mechanics layer with a data-driven neural network to estimate grasp compliance and in-hand tool displacement for an underactuated three-finger hand.
*   **[GraspADMM: Improving Dexterous Grasp Synthesis via ADMM Optimization](https://arxiv.org/abs/2603.13832)**: Proposes GraspADMM, a dexterous grasp synthesis framework that uses Alternating Direction Method of Multipliers (ADMM) to decouple contact points on the object from contact locations on the hand, resulting in a nearly 15% improvement in grasp success rate over prior state-of-the-art.
*   **[Learning Stable In-Grasp Manipulation in a Non-Dropping Action Space](https://arxiv.org/abs/2606.28196)**: Decomposes dexterous manipulation skills into analyzable components and uses RL with constraints and guidance from classical physics and control theory for efficient, stable in-grasp repositioning and reorientation without dropping objects.
*   **[Seg2Grasp: A Robust Modular Suction Grasping in Bin Picking](https://arxiv.org/abs/2607.17757)**: A modular pipeline (Segmentation, Grasping, Classification) for robust suction grasping in cluttered bins. It uses a Transformer-based model for class-agnostic object masks and fine-tuned Mask-CLIP for object identification.
*   **[GenHOI: Contact-Aware Humanoid-Object Interaction by Imitating Generated Videos without Task-Specific Training](https://arxiv.org/abs/2606.12995)**: Enables zero-shot humanoid-object interaction by imitating single generated videos. The framework reconstructs the scene, generates an interaction video, and extracts contact regions for optimization-based trajectory refinement.

### Vision-Language-Action (VLA) Models
*   **[Patch Policy: Efficient Embodied Control via Dense Visual Representations](https://arxiv.org/abs/2607.18236)**: Introduces Patch Policy, a lightweight architecture that consumes dense pre-trained patch tokens directly from ViTs, achieving a 40% relative improvement over policies using global-pooled representations while using only 0.7% of the parameters of full VLAs.
*   **[FM-VLA: Force-based Memory for Vision-Language-Action Models in Contact-Rich Manipulation](https://arxiv.org/abs/2607.18231)**: Proposes FM-VLA, augmenting VLA models with force-based memory tokens to handle non-Markovian, contact-rich tasks. It achieves over 80% success rate on memory-dependent tasks like button pressing and wiping.
*   **[PrimitiveVLA: Learning Reusable Motion Primitives for Efficient and Generalizable Robotic Manipulation](https://arxiv.org/abs/2605.28634)**: Shifts from direct instruction-to-control mapping to a primitive-centric paradigm. It disassembles demonstrations into reusable motion primitives and assembles them for robust, generalizable manipulation.
*   **[TempoVLA: Learning Speed-Controllable Vision-Language-Action Policies](https://arxiv.org/abs/2606.06491)**: A single VLA that allows explicit control of execution speed. Uses a Variable-Speed Trajectory Augmentation (VSTA) and a conditioning mechanism to decouple speed from actions.
*   **[Semantic Anchoring for Robotic Action Representations](https://arxiv.org/abs/2607.13597)**: Investigates the erosion of semantic structure in VLA fine-tuning and proposes a plug-and-play method that anchors action representations to a semantic manifold, improving out-of-distribution generalization by up to 21.5%.
*   **[Reasoning as a Double-Edged Sword: Architecture and Cross-Stage Robustness in Vision-Language-Action Models](https://arxiv.org/abs/2607.17786)**: Investigates the robustness of VLA models to perturbations. Finds that a latent-iterative reasoning model (like Diffusion Policy) is far less robust than chain-of-thought or no-reasoning models under both stochastic and white-box noise.
*   **[Closing the Loop in Humanoid VLA: Persistent 3D Object Tokens for Verifiable Loco-Manipulation](https://arxiv.org/abs/2607.18016)**: Proposes Persistent Object Tokenization (POT) to maintain persistent 3D object records for closed-loop humanoid loco-manipulation, significantly improving task success on a Unitree G1 over a matched baseline.
*   **[RobustVLA: On Robustness of Vision-Language-Action Model against Multi-Modal Perturbations](https://arxiv.org/abs/2510.00037)**: Evaluates VLA robustness against 17 perturbations across 4 modalities. Proposes RobustVLA with offline robust optimization and input consistency, achieving significant gains over baselines.
*   **[ConceptTree: Bringing Semantic Transparency to Black-Box Decision Making for Robotic Manipulation](https://arxiv.org/abs/2607.17861)**: A framework for interpretable high-level skill selection in manipulation by representing policies as a decision tree over human-interpretable concepts, enabling traceable and intervenable decisions.

### Tactile Sensing & In-Hand Manipulation
*   **[Feeling the Unexpected: ResTacVLA for Contact-Rich Manipulation via Residual Tactile Representation](https://arxiv.org/abs/2607.03387)**: Integrates tactile feedback into VLAs by reformulating it as a "residual tactile representation" (inspired by Predictive Coding) to capture discrepancies between visual priors and physical sensations, preventing visual dominance in contact-rich tasks.
*   **[A Sensorless, Inherently Compliant Anthropomorphic Musculoskeletal Hand Driven by Electrohydraulic Actuators](https://arxiv.org/abs/2603.24357)**: A fully sensorless robotic hand powered by Peano-HASEL actuators that uses the actuators' self-sensing properties for closed-loop contact-aware control, enabling inherently safe grasping of fragile objects.
*   **[Stochastic Entanglement of Deterministic Origami Tentacles For Robust Robotic Grasping](https://arxiv.org/abs/2604.26897)**: A tendon-driven origami tentacle gripper that exploits stochastic entanglement (braiding, knotting) for robust grasping of random-shaped objects with simple design and actuation.
*   **[Continuously Stable Structure through Plastic Deformation](https://arxiv.org/abs/2607.15659)**: Introduces a soft gripper that uses plastic deformation to maintain stable, continuous configurations without energy input, enabling passive holding forces up to 16 N and stable perching.

### Teleoperation & Data Collection
*   **[Real-Time sEMG-Based Telecontrol of an Assistive Robotic Arm Using a 1D Convolutional Neural Network](https://arxiv.org/abs/2607.16310)**: A real-time sEMG interface using a 1D CNN for teleoperation of an assistive robotic arm. Achieves >90% classification accuracy and ~0.32s latency.
*   **[MEVION: Low-Cost Open-Source Data Collection System for Powerful and High-Speed Dual-Arm Manipulation](https://arxiv.org/abs/2607.17970)**: Presents MEVION, a low-cost (~$14,000), open-source dual-arm robot data collection system capable of generating higher force and speed than ALOHA, enabling imitation learning for new manipulation tasks.
*   **[Open-AoE: An Open Egocentric Manipulation Dataset and Toolchain for Embodied Learning](https://arxiv.org/abs/2607.14183)**: A large-scale (approx. 2000 hours) egocentric manipulation dataset from 500+ contributors, with a toolchain for processing and downstream use in training VLA policies and World Models.

### Sim-to-Real Transfer & World Models
*   **[World Translation: Minimizing Sim-to-Real Gap with Backward Dynamics Extraction and Unpaired Domain Translation](https://arxiv.org/abs/2607.18154)**: Proposes "World Translation" to address the sim-to-real gap by learning a backward dynamics model that extracts unobservable dynamics information, then translating it across simulation and reality as an unpaired domain-translation problem.
*   **[Isaac Sim-to-Real: Reinforcement Learning based Locomotion for Quadrupeds](https://arxiv.org/abs/2607.18135)**: Demonstrates zero-shot sim-to-real transfer of a RL-based locomotion policy for the Unitree Go1, trained in Isaac Sim and achieving velocities comparable to the integrated controller with better disturbance recovery.
*   **[SoMA: A Real-to-Sim Neural Simulator for Robotic Soft-body Manipulation](https://arxiv.org/abs/2602.02402)**: Presents SoMA, a 3D Gaussian Splatting simulator for soft-body manipulation that couples deformable dynamics, robot actions, and environmental forces in a unified latent neural space for stable long-horizon simulation.
*   **[Learning Adaptive Safety Margins for Visual Navigation](https://arxiv.org/abs/2607.18200)**: Proposes a context-conditioned safety critic that learns an adaptive clearance preference for navigation trajectory selection, demonstrating zero-shot transfer from simulation to a Unitree G1 humanoid.

### Robotics Systems & Control
*   **[S.E.A.G.R: A Socially and Emotionally Aware Greeting Robot Framework with Dual-Layer Cultural and Affective Modulation](https://arxiv.org/abs/2607.16341)**: A greeting robot framework (SEAGR) that adapts behavior based on user culture and emotion, using a dual-layer modulation approach and a Sense-Think-Act architecture.
*   **[Control Design for a Rideable Animatronic Two-Wheeled Robot with Quadruped Form](https://arxiv.org/abs/2607.16302)**: A rideable animatronic robot with limbs for expressive movement and a self-balancing two-wheeled base for efficient locomotion. Focuses on robust self-balancing control and generating natural quadruped-like motion.
*   **[PhyAgentOS: A Self-Evolving Operating System for Embodied Agents with Decoupled Cognitive Planning and Physical Execution](https://arxiv.org/abs/2607.16636)**: Proposes PhyAgentOS, a runtime OS for embodied agents that decouples cognitive planning from physical execution using a file system interface and a session-centered verification approach.
*   **[Token-Wise Latent Streaming from Slow Reasoners to Fast Planners for Dynamic Vision Language Navigation](https://arxiv.org/abs/2607.16806)**: Addresses latency in VLM-based navigation by streaming intermediate VLM hidden states to a fast flow-matching planner, mitigating staleness and improving social navigation in dynamic environments.
*   **[SAGE: A Socially-Aware Generative Engine for Heterogeneous Multi-Agent Navigation](https://arxiv.org/abs/2607.16619)**: A diffusion-based framework (SAGE) for heterogeneous multi-agent navigation that encodes asymmetric interactions via a Heterogeneous Graph Transformer and refines trajectories with safety-social energy guidance.
*   **[Articulated Humanoid Head for a Robot Receptionist Capable of Natural Human Interaction](https://arxiv.org/abs/2607.17042)**: A 21-DoF humanoid robot head with realistic skin for a receptionist role, integrating face recognition, NLP, and emotional expression.
*   **[Learning to Fold: prizewinning solution at LeHome Challenge 2026](https://arxiv.org/abs/2606.27163)**: Describes the winning solution for bimanual garment folding, which improves a VLA policy with a reinforcement-learning loop where the policy acts as its own value function.
