# 今日 arXiv 机器人学与人工智能论文精选

## 灵巧操作与手部操控

**[HANDFUL: Sequential Grasp-Conditioned Dexterous Manipulation with Resource Awareness](https://arxiv.org/abs/2604.25126)**
研究灵巧手在序列化多技能操作中的"资源感知"抓取策略。核心洞察是：在连续操作任务中，手指是一种有限资源，需要为后续动作保留。HANDFUL 框架通过手指级接触奖励促使策略探索资源感知抓取，并采用课程式策略学习来优化下游任务表现。实验在仿真和真实 LEAP 手上验证了该方法相比贪心抓取策略的显著优势。

**[LAMP: Latent Motion Prior-Guided Real-World Learning for Dexterous Hand Manipulation](https://arxiv.org/abs/2607.06323)**
提出潜层运动先验模块，将手部动作历史映射为紧凑的先验分布，并解码为可执行的高维手部目标。基于此构建的三阶段真实世界学习框架（预训练先验、训练视觉运动策略、在线残差RL）在四个灵巧操作任务中实现平均98.75%成功率。关键创新在于共享的可解码接口使残差探索集中在演示接触一致性动作附近。

**[Do Robots Really Need Anthropomorphic Hands?](https://arxiv.org/abs/2508.05415)**
系统综述性论文，分析人类手部特征与商用机器人手部的对比。通过对125篇论文的文献分析发现：复杂五指手并非所有任务必需，手内操作并不受益于拟人化设计；但机制复杂度与操作任务广度相关。文章呼吁建立标准化评估标准以系统地比较手部设计。

## 遥操作与人类示教

**[DexTele: A Dual-Arm Dexterous Teleoperation System](https://arxiv.org/abs/2607.05883)**
提出双灵巧手遥操作系统DexTele，包含基于视觉的运动重定向模块和自适应抓取模块。运动重定向采用运动-图编码器和潜层优化实现跨平台精准重定向；自适应抓取结合视觉语言模型与模型预测控制预测所需抓取力并进行在线优化。

**[RynnWorld-Teleop: An Action-Conditioned World Model for Digital Teleoperation](https://arxiv.org/abs/2607.06558)**
提出"数字遥操作"新范式：用生成式世界模型替代真实机器人，操作员的手势流驱动模型合成高保真视频，记录的姿势流作为与具体硬件无关的动作标签。40+ FPS实时交互生成，仅基于生成数据训练的策略实现零样本Sim2Real迁移。

**[WristMimic: Full-Body Humanoid Control with Wrist-Guided Manipulation](https://arxiv.org/abs/2607.06438)**
提出手腕引导的全身控制框架，明确分离无接触的身体运动与接触密集的手部操作。关键洞察是手腕作为两个区域之间的自然"门"，可运动学跟踪却决定全局手部配置。不直接监督手指，而是通过物体跟踪和接触结果学习抓取行为。

**[HumanoidUMI: Bridging Robot-Free Demonstrations and Humanoid Whole-Body Manipulation](https://arxiv.org/abs/2606.27239)**
受UMI启发，提出便携式无机器人框架HumanoidUMI，使用轻型VR设备和UMI启发式夹爪收集人体关键点轨迹、腕部观察和夹爪动作。训练高级策略预测未来关键点并重定向为机器人全身控制参考。

## 触觉感知与操作

**[Feeling the Unexpected: ResTacVLA for Contact-Rich Manipulation via Residual Tactile Representation](https://arxiv.org/abs/2607.03387)**
受预测编码机制启发，将触觉数据重定义为"残差触觉表示"——捕捉视觉先验与物理感知之间的差异。通过VQ瓶颈离散化为潜层接触基元，并利用视觉先验的不确定性自适应门控触觉整合。在一系列接触密集操作任务中持续优于基线方法。

## VLA模型与世界模型

**[Learning 4D Geometric Priors for Inference-Efficient World Action Models](https://arxiv.org/abs/2607.05468)**
提出MECo-WAM，通过多专家协同训练注入4D几何先验到视频-动作表示中，同时保持轻量级推理图。采用衰减式4D读掩码注意力实现几何知识迁移，在LIBERO(98.2%)和RoboTwin 2.0(92.6%)上取得优异结果且不增加推理成本。

**[LAM: 4D Embodied World Models for Robotic Manipulation](https://arxiv.org/abs/2607.06559)**
论证RGB、深度和光流的同步多模态(称为RGB-DF)提供了物理场景4D动力学的表示，比2D像素视频更接近机器人需的低级末端执行器动作。提出统一扩散框架同时预测未来RGB帧、深度图和光流，并设计策略头消费4D表示实现闭环控制。

**[WAM4D: Fast 4D World Action Model via Spatial Register Tokens](https://arxiv.org/abs/2606.14048)**
使用轻量级空间注册令牌作为训练时的未来深度读出器，将预训练几何先验迁移到因果视频-动作变换器中，推理时移除注册分支实现高效动作生成。

**[FORGE: Towards Functional Tool-Use Generalization via Keypoint Trajectory Reasoning](https://arxiv.org/abs/2607.05780)**
研究机器人功能工具通用化问题——在未见过的工具上泛化使用功能。发现关键点轨迹是最佳中间表示，平衡功能表达性和动作可落地性。提出两阶段策略：从无动作数据预测可泛化关键点轨迹，再用有限演示将其落地为机器人动作。在7种工具的敲击功能基准上实现2倍以上平均成功率提升。

**[From Foundation to Application: Improving VLA Models in Practice](https://arxiv.org/abs/2607.06403)**
提出LingBot-VLA 2.0，在三个方向改进：跨任务和跨本体泛化(6万小时训练数据含20种机器人配置)；扩展动作空间(包含头部、腰部、移动基座和灵巧手自由度)；通过视频表示模型提供的语义先验进行预测性动力学建模。

## 抓取与操作规划

**[GraspIT: A Dataset Bridging the Sim-to-Real Gap for Validated SE(3) Grasp Generation](https://arxiv.org/abs/2607.05869)**
提出连接仿真与真实世界的抓取数据集，通过四阶段物理滑移测试标注，提供轨迹可达性检查和连续质量评分。包含约2.3M候选抓取，反转标签到100个真实场景，支持桌面操纵策略学习和行为克隆。

**[First Plan Then Evaluate: Multi-Target Planning with Post-Planning Success Evaluation](https://arxiv.org/abs/2509.07162)**
提出先生成候选抓取并规划轨迹，再评估最优抓取，而非传统的"生成-评估-规划"串行流程。实验表明该方法在不同物体、生成器和运动规划器上均优于传统框架，并成功泛化到含不同架子和桌面高度的真实环境。

## 仿真到真实迁移

**[Actuator Reality Shaping for Zero-Shot Sim-to-Real Robot Learning](https://arxiv.org/abs/2607.02205)**
提出"致动器现实塑造"新范式：不修改仿真器以匹配真实世界，而是塑造物理致动器的闭环行为以匹配仿真中使用的理想二阶参考动力学。通过两自由度前馈-反馈控制器实现零样本真实部署，在单关节伺服、7自由度机器人臂、轮腿机器人和人形机器人上验证。

**[Image2Sim: Scaling Embodied Navigation via Generative Neural Simulator](https://arxiv.org/abs/2607.05765)**
提出实时神经仿真框架，从姿态RGB-D图像序列构建高质量互动环境。核心思路是解耦3D空间锚定与真实感观察合成。将大规模视频/图像集合转化为近20K互动场景和超过1千万导航训练样本，训练的策略在现实零样本场景中有效迁移。

## 物体投掷与操作

**[Learning to Throw Objects Safely in Multi-Obstacle Environments](https://arxiv.org/abs/2607.06388)**
解决在杂乱环境中向目标篮子投掷物体同时避开障碍物的问题。引入势场状态表示紧凑编码篮子吸引力和障碍物排斥力。基于SAC的策略在仿真中训练，真实机器人实验实现高达90%成功率，展示了从仿真到真实的鲁棒迁移。

## 数据集与基准

**[RoboDojo: A Unified Sim-and-Real Benchmark for Comprehensive Evaluation of Generalist Robot Manipulation Policies](https://arxiv.org/abs/2607.04434)**
提出统一仿真与真实评估基准，包含42个仿真任务和18个真实任务覆盖五种能力维度。支持并行仿真环境实现可扩展评估，并集成30个策略建立公有排行榜。

**[EgoVerse: An Egocentric Human Dataset for Robot Learning from Around the World](https://arxiv.org/abs/2604.07607)**
协作式人类数据驱动的机器人学习平台，包含1,362小时(8万episode)人类示教，覆盖1,965个任务、240个场景和2,087名示教者。大规模研究人类到机器人迁移，发现策略性能通常随人类数据增加而提升，但有效扩展取决于人类数据与机器人学习目标的对齐。

## 其他值得关注的工作

**[RoboTALES: Learning Reasoning-Guided Robot Policies via Task-Aligned Simulated Futures](https://arxiv.org/abs/2607.06018)**
利用分层LLM规划器将复杂任务分解为子目标序列引导世界模型想象，结合VLM批评家评估"想象"的未来并给予奖励反馈。在机器人操作任务中持续优于现有方法，尤其在长时域任务中。

**[Optimal Transport Q-Learning for Flow Policy Steering and Acceleration](https://arxiv.org/abs/2607.06262)**
提出OTQL，通过优势加权条件最优传输流匹配微调和加速流策略。仅需50-60个episode互动预算即可将单任务策略从36%提升至86%成功率，同时减少70%动作生成推理步数。

**[Training-Free Acceleration for Vision-Language-Action Models with Action Caching and Refinement](https://arxiv.org/abs/2607.06370)**
提出ActionCache，即插即用的外部缓存机制，重用过去中间动作来预热生成，大幅降低推理延迟。对π0.5和GR00T-N1.6分别实现11.75倍和34.43倍推理加速。

**[Structure-Aware Data Selection for Imitation Learning with VLA Models](https://arxiv.org/abs/2607.06442)**
提出SIEVE，将示教分解为可复用的基元和转换接口，通过结构暴露感知的资源分配选择最具代表性的示教。仅用50%示教和50%训练步数即可超越全数据训练。
