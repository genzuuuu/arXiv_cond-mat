# Today's arXiv Papers: Robotics & AI Feeds

## Dexterous Manipulation & Multi-Finger Grasping

### [DexCompose: Reusing Dexterous Policies for Multi-Task Manipulation with a Single Hand](https://arxiv.org/abs/2606.28323)
A role-aware residual composition framework that reuses pretrained dexterous policies for multi-task manipulation. Uses explicit finger-level action ownership to avoid destructive interference when adding new tasks. Achieves 77.4% composite success rate across 16 tasks.

### [RetrDex: Efficient Object Retrieval in Cluttered Scenes with a Dexterous Hand](https://arxiv.org/abs/2502.18423)
An efficient framework for dexterous arm-hand systems to retrieve objects buried under clutter. Uses large-scale parallel RL with spatially aware representations encoding occlusion patterns. Demonstrates zero-shot transfer to real-world dexterous multi-fingered systems.

### [Grasp-Oriented Non-Prehensile Manipulation via Learning a Graspability Field](https://arxiv.org/abs/2606.30474)
Reformulates non-prehensile manipulation for grasping as optimizing an object-centric graspability objective rather than reaching a specific pose. Constructs a graspable set and uses RL to reconfigure objects into graspable states without external planners.

## Tactile Sensing & Visuo-Tactile Policies

### [TACO: A Test and Check Framework for Robust Pose Graph Optimization](https://arxiv.org/abs/2606.29851)
Presents a test-and-check optimization framework for filtering outliers in SLAM. Achieves 90%+ success rates across outlier rates up to 50% with mean convergence times of ~45-100ms.

### [TAP-VLA: Tactile Annotation Prompting for Vision Language Action Models](https://arxiv.org/abs/2606.29089)
Supplies tactile feedback through visual augmentation rather than architectural change. Extracts shear fields from visuo-tactile sensors and overlays them as spatially-grounded vectors onto RGB images. Achieves 78% success on contact-rich tasks vs. under 50% for vision-only baselines.

### [TacGen: Touch Is a Necessary Dimension of Physical-World Representation](https://arxiv.org/abs/2606.29173)
Mitigates tactile-data scarcity by combining V+T contrastive alignment with a latent-space V→T generator. V+T improves on mass (ΔR²=+0.570), density (+0.067), hardness (+0.117), and force labels (+0.281) over vision-only.

### [Seeing Touch from Motion: Visuo-Tactile Policy with Tactile Motion Correlation](https://arxiv.org/abs/2606.29941)
Proposes a motion-aware tactile representation using correlation between transient and cumulative motion to distinguish fine-grained contact states. Uses Mixture-of-Transformers for unified modality-aware visuo-tactile policy.

### [Heterogeneous Tactile Transformer](https://arxiv.org/abs/2606.29948)
Learns shared tactile representations across heterogeneous sensors. Uses sensor-specific encoders, shared transformer trunk, and cross-modal alignment. Release HPT dataset with 1.6M synchronized paired frames across four sensor types.

### [YOTO: You Only Touch Once - 6-DoF Object Pose Estimation from Single Tactile Contact](https://arxiv.org/abs/2606.28899)
A tactile-only pose estimation system recovering full 6-DoF object pose from a single pair of simultaneous contacts. Uses coarse-to-fine network for contact localization and closed-form normal-aware SVD solver for pose recovery.

### [TacEvo: Self-Evolving Architecture Discovery for Robotic Tactile Perception](https://arxiv.org/abs/2606.30109)
LLM-driven quality-diversity search for tactile neural architectures. Uses MAP-Elites loop with code-level mutations. Achieves 96.0%/94.5% autonomous generation reliability and improves fitness by 56.1%/96.1%.

## Visuomotor Policies & VLA Models

### [SA-VLA: State-aware tokenizer for improving VLA models' performance](https://arxiv.org/abs/2606.30113)
A state-aware action tokenizer that conditions action decoding on robot state. On 12 RoboTwin tasks, improves average success rate from 0.29 to 0.56 over strongest tokenizer baseline.

### [T^2VLA: Confidence-Driven Test-Time RL for VLA Models](https://arxiv.org/abs/2606.29892)
Enables VLA self-bootstrapping improvement using trajectory-level similarity to high-confidence demonstrations as intrinsic reward. Outperforms supervised baselines on LIBERO and RoboTwin without external reward feedback.

### [X-Tokenizer: A Multimodal Action Tokenizer for VLA Pretraining](https://arxiv.org/abs/2606.14752)
Formulates action tokenization as semantic interface learning. Uses Semantic Residual Quantization with Masked Action Modeling. Outperforms FAST in multimodal grounding (+13.5%) and long-horizon tasks (+8.25%).

### [ZR-0: Training VLAs with Dense Embodied Chain-of-Thought Supervision](https://arxiv.org/abs/2606.30552)
Uses dense Embodied Chain-of-Thought supervision to align cross-embodiment representations. Dual-stream architecture with VLM generating ECoT reasoning and Diffusion Transformer for actions. Pre-trained on ~60M frames.

### [Behavior Prompting Policy: Demonstrations as Prompts for Manipulation](https://arxiv.org/abs/2606.30457)
Enables robots to perform new tasks at inference time given a single human demonstration. Uses in-context visuomotor architecture with a handheld manipulation interface (iPhUMI) for diverse data collection.

### [MoRE: Behavior Uncloning - Distilling Mode Redirection into Policy Weights](https://arxiv.org/abs/2606.29201)
Redirects policy rollouts toward desired behavior modes through a short "uncloning" step. Distills redirection signal from a temporary mode classifier into policy weights. Improves success rate by 44 percentage points over mixed-mode policies.

### [S²-VLA: State-Space Guided VLA for Long-Horizon Manipulation](https://arxiv.org/abs/2606.27872)
Introduces State-Space Guided Adaptive Attention that maintains a belief state tracking task progression. Generates dynamic gating weights for adaptive feature fusion. Outperforms 7B-scale models despite 2B parameter size.

### [X-DiffVLA: Diffusion Action Heads for VLA Models](https://arxiv.org/abs/2606.25044)
A diffusion-based VLA model for cross-embodied settings. Uses Embodiment Forcing and Morphological Tree Diffusion. Achieves improvements of 15.3% and 12.5% on RoboCasa and Isaac Gym across grippers to dexterous hands.

### [VLA Model Deployment: Experimental Insights from a UR5 Platform](https://arxiv.org/abs/2606.30456)
Investigates transfer of VLA models to real-world UR5e. Key finding: successful deployment depends less on model capacity and more on precise control of the entire data-model-control pipeline.

### [Drop-Then-Recovery: How Redundant Are VLA Models?](https://arxiv.org/abs/2606.27755)
Studies architectural redundancy in VLA models using transformer block removal. Finds language backbones highly redundant: removing half LLM blocks even improves OpenVLA-OFT from 95.0% to 98.3%.

### [Event-VLA: Action-Conditioned Event Fusion for Robust VLA Model](https://arxiv.org/abs/2606.29384)
Event-enhanced VLA framework for manipulation under varying illumination. Uses event streams as illumination-robust complementary observation with action-query routing pathway.

### [CORE: Common Outcome Regularities from Action-Free Visual Demonstrations](https://arxiv.org/abs/2606.29517)
Extracts Common Outcome Regularities from visual demonstrations without robot executable actions. Uses terminal outcome encoder and visual goal prototypes. Improves success rate by up to +17.0 percentage points.

### [Chronos: A Physics-Informed Full-History Framework for Non-Markovian Manipulation](https://arxiv.org/abs/2606.30318)
Elevates observation history to the latent state of policy dynamics. On RMBench memory-dependent tasks achieves 73.6% vs pi0.5's 11.2% with 10x fewer parameters.

## Teleoperation & Human-to-Robot Transfer

### [Human2Any: Human-to-Robot Transfer via Constraint-Aware Compositional Planning](https://arxiv.org/abs/2606.28813)
Learns reusable object-centric interaction priors from human videos. Composes learned priors with robot-side feasibility reasoning. Validated on Franka and RBY-1 humanoid without real-world robot training data.

### [WARP: Whole-Body Retargeting for Learning from Offline Human Demonstrations](https://arxiv.org/abs/2606.29940)
Offline pipeline for extracting precise whole-body actions from human demonstrations. Uses closed-form Shoulder-Elbow-Wrist geometric solver. First framework for zero-shot whole-body mobile manipulation from offline human demonstrations.

### [Keypose Exploration: Automatic Trajectory Labelling and Cross-Embodiment Policy Transfer](https://arxiv.org/abs/2606.29028)
Automatic trajectory labelling pipeline combining VLMs for semantic detection with trajectory analysis. Uses keypose-guided Diffusion Policy with reachability filtering for cross-embodiment transfer.

### [X-Morph: Human Motion Priors for Scalable Robot Learning Across Morphologies](https://arxiv.org/abs/2606.30290)
Converts human motion into deployable policies for diverse non-humanoid morphologies. Evaluated on quadruped, hexapod, and quadruped-manipulator platforms.

### [ConCent: Contact-Centric Real-to-Sim-to-Real Learning from One Demonstration](https://arxiv.org/abs/2606.30268)
Uses task-relevant contact event sequences from real demonstrations as learning objective. Optimizes contact geometry in simulation for dynamics matching. Provides structured reward signal preventing unrealistic simulator contacts.

### [Grounding Sim-to-Real Generalization in Robotic Manipulation](https://arxiv.org/abs/2603.22876)
Empirical study across multi-level domain randomization, photorealistic rendering, physics-realistic modeling, and RL updates. Over 10k real-world trials deriving critical insights for sim-to-real transfer.

### [PhysisForcing: Physics Reinforced World Simulator for Robotic Manipulation](https://arxiv.org/abs/2606.28128)
Strengthens physical consistency in video generation by focusing supervision on physics-informative regions. Improves Wan2.2-I2V-A14B and Cosmos3-Nano by 22.3% and 9.2% on R-Bench.

### [WoVR: World Models as Reliable Simulators for Post-Training VLA Policies with RL](https://arxiv.org/abs/2602.13977)
Reliable world-model-based RL for VLA post-training. Uses controllable action-conditioned video world model, keyframe-initialized rollouts, and co-evolution to handle hallucination.

### [ReactiveBFM: Reactive Closed-Loop Motion Planning for Universal Humanoid Whole-Body Control](https://arxiv.org/abs/2606.30362)
Real-time closed-loop planning-control framework for humanoids. Uses scheduled prefix sampling curriculum and asynchronous replanning. Achieves 93.1% success rate under severe perturbations.

### [SceneBot: Contact-Prompted Humanoid Whole Body Tracking with Scene-Interaction](https://arxiv.org/abs/2606.27581)
Unified motion-tracking framework for free-space locomotion, terrain traversal, and whole-body manipulation. Uses hindsight scene reconstruction for contact-rich data. First framework to unify free-space and contact-rich behaviors.

### [VLK: Learning Humanoid Loco-Manipulation from Synthetic Interactions](https://arxiv.org/abs/2606.30645)
Generates vision-language-kinematics supervision synthetically in reconstructed scenes. Produces 48,000 paired trajectories. Evaluated on Unitree G1 for navigation and object transport.

### [FADA: Few-Shot Domain Adaptation via Dynamics Alignment for Humanoid Control](https://arxiv.org/abs/2606.28476)
Planner-IDM framework for few-shot adaptation using ~2 minutes target-domain rollouts. Outperforms in-context and end-to-end adaptation baselines on diverse whole-body tasks.

### [WOLF-VLA: Whole-Body Humanoid Optimal Locomotion Framework for VLA](https://arxiv.org/abs/2606.25591)
Integrates whole-body optimal-control motion synthesis with multi-modal dataset for VLA-based locomotion policies. Covers six locomotion task families with natural-language instructions.

### [FT-WBC: Learning Fault-Tolerant Whole-Body Control for Legged Loco-Manipulation](https://arxiv.org/abs/2606.24466)
Fault-tolerant loco-manipulation framework with Fault Estimator and Posture Adaptation Module. Zero-shot transfer to real legged manipulator under actuator failures.

### [FutureNav: Unified World-Action Modeling for VLN](https://arxiv.org/abs/2606.30367)

### [Qwen-RobotNav: A Scalable Navigation Model for Agentic Navigation Systems](https://arxiv.org/abs/2606.18112)

### [DRIVE-Nav: Directional Reasoning for Open-Vocabulary Navigation](https://arxiv.org/abs/2603.28691)

### [CORE Planner: Contextual-memory Oriented RL for Robot Navigation](https://arxiv.org/abs/2606.29222)

### [MOAR Planner: Multi-Objective Adaptive Risk-Aware Path Planning for UAV Inspection](https://arxiv.org/abs/2606.30575)
Risk-aware path planner integrating pre-computed cost maps, damage/insertion costs, and adaptive speed planning. Evaluated through simulations and real-world flight tests.

### [LNN-Fly: Continuous-Time UAV Navigation under Timing Mismatch](https://arxiv.org/abs/2606.28827)

### [SPARK: Sequential Planning via Anchored Robotic Keypoints](https://arxiv.org/abs/2606.30613)

### [GROW²: Grounding Which and Where for Robot Tool Use](https://arxiv.org/abs/2606.30632)

### [SSI-Policy: Learning Structured Scene Interfaces for Vision-Language Robotic Manipulation](https://arxiv.org/abs/2606.26800)

### [STEAM: Self-Supervised Temporal Ensemble Advantage Modeling](https://arxiv.org/abs/2606.29834)
Learns frame-level advantages from expert demonstrations without labels. Improves policy success rate by 59%, 54.3%, 23% and 16.2% across four real-world tasks.

### [CI-MSE: Critical Interval MSE for Offline Validation of Manipulation Policies](https://arxiv.org/abs/2606.29898)
