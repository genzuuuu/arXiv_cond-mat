### Dexterous Manipulation & Grasping

*   **[DynaMOMA: Instantaneous Prediction of Grasp Poses for Mobile Manipulation of Dynamic Objects](https://arxiv.org/abs/2606.25295)**
    This paper tackles the challenging task of mobile manipulation of moving objects. It introduces a framework that couples an anchor-based diffusion model for predicting temporally consistent grasp trajectories with a whole-body reinforcement learning policy. A key contribution is the anticipation-guided reward, which helps the policy dynamically shift its target from the current grasp observation to the predicted trajectory, enabling effective grasping of dynamic objects in both simulation and real-world experiments.

### Tactile Sensing & Perception

*   **[TacVerse: A Multi-Sensor Dataset and Benchmark for Cross-Sensor Vision-Based Tactile Perception](https://arxiv.org/abs/2606.25877)**
    This paper presents TacVerse, a comprehensive multi-sensor dataset and benchmark for vision-based tactile sensors (VBTSs). It contains 106,800 tactile images from seven different VBTSs for tasks like shape classification, grating classification, and force regression. The work provides a controlled testbed to study sensor shift and cross-sensor generalization, finding that while within-sensor performance is strong, direct cross-sensor transfer degrades significantly, but few-shot adaptation can help.

*   **[ForceBand: Learning Forceful Manipulation with sEMG](https://arxiv.org/abs/2606.26093)**
    ForceBand introduces a low-cost, wrist-worn sEMG system that enables human demonstrators to provide force-enriched demonstrations for robot learning. By pre-training an EMG2Force model on a multimodal dataset (sEMG, IMU, video, fingertip forces), the system can predict per-finger forces from sEMG alone after a short calibration. This allows non-experts to demonstrate force-sensitive tasks, significantly improving policy success on tasks requiring object-specific force control.

### Visuomotor Policies & Imitation Learning

*   **[Memory Retrieval in Visuomotor Policies for Long-Horizon Robot Control](https://arxiv.org/abs/2606.25136)**
    This paper introduces HALO, a visuomotor policy with an attention-based memory retrieval mechanism for long-horizon control in partially observable environments. To avoid learning spurious correlations and mitigate error accumulation, HALO uses a VLM to distill memory-dependent reasoning into the policy and employs sparse attention to retrieve only the most relevant history. It successfully handles tasks requiring recall of events from up to eight minutes in the past.

*   **[Decoupling Semantics and Geometric Grounding: Spatial Visual Prompts for Language-Conditioned Imitation Learning](https://arxiv.org/abs/2606.25360)**
    This work proposes SVP-IL, a decoupled architecture for language-conditioned imitation learning. It separates the semantic reasoning of a VLM from the geometric action generation. The VLM parses language instructions into explicit "Spatial Visual Prompts" (zero-shot geometric masks), which are then injected into a lightweight action generator via direct feature-level fusion. This leads to significant improvements in data efficiency and performance on ambiguous, language-conditioned tasks, especially with few demonstrations.

*   **[RoboRouter: Training-Free Policy Routing for Robotic Manipulation](https://arxiv.org/abs/2606.25241)**
    RoboRouter is a training-free framework that acts as a "router" for a pool of heterogeneous robot policies (VLAs, VA policies, code-based). Instead of proposing a single new policy, it learns to select the best-performing policy for a given task based on a semantic task representation and historical execution records. This approach leverages the complementary strengths of existing policies, improving average success rates without additional training overhead.

*   **[GRAFT: Graph-Based Affordance Transfer via Part Correspondence](https://arxiv.org/abs/2606.25241)**
    GRAFT tackles the problem of zero-shot manipulation transfer to unseen objects using a single demonstration. It represents objects as part-based graphs, using part-level descriptors for instance retrieval and fine-grained contact point matching. This geometry-aware framework outperforms semantic-only baselines for transferring manipulation affordances to novel objects.

### Sim-to-Real & Domain Adaptation

*   **[AISPO: Enhancing Depth Reliability for Robotic Manipulation of Non-Lambertian Objects via Affine-Invariant Shape Prior](https://arxiv.org/abs/2606.25503)**
    This paper addresses depth perception failures for non-Lambertian (transparent/specular) objects. AISPO is a depth completion framework that uses multi-scale RGB-D feature fusion with an affine-invariant shape prior to enforce geometric consistency. The improved depth reliability, which emphasizes physical plausibility, significantly boosts real-world grasping success rates for these challenging objects.

*   **[A 3D-Printable Dataset for Fair Testing and Comparisons of Tactile Sensors](https://arxiv.org/abs/2606.25886)**
    This paper introduces the first physically reproducible, open-source dataset of 3D-printable textures for tactile sensor benchmarking. The textures are based on mathematical functions, allowing for controlled variation. Results from a TacTip sensor show that print quality affects tactile variance, but high-quality prints support strong within-printer generalization, providing a foundation for fairer comparisons across different tactile sensors.

*   **[DeformGen: Dynamics-Based Topology Augmentation for Deformable Manipulation Policy Learning](https://arxiv.org/abs/2606.25939)**
    This paper presents a demonstration augmentation framework specifically for deformable objects. DeformGen tackles the challenge of generating diverse, valid configurations by applying localized physical disturbances in simulation. It also adapts the end-effector trajectory for new states using deformation-field warping. This method proves more effective for learning deformable manipulation policies than standard rigid-style augmentation.

### Humanoids & Locomotion (Brief Note)

*   **[RGB: RL Guided Whole-Body MPPI for Humanoid Control](https://arxiv.org/abs/2606.25123)**
    This work combines an RL prior with a Model Predictive Path Integral (MPPI) controller. The RL policy provides a sampling prior for dynamic feasibility, while MPPI corrects drift to satisfy task-specific objectives. This framework achieves stable, high-rate control on a 29-DoF humanoid, improving precision over a pure RL baseline.
