### Tactile Sensing & Visuo-Tactile Learning

- **[TactX: Learning Shared Tactile Representations Across Diverse Sensors](https://arxiv.org/abs/2606.31236)**
  Proposes TactX, a framework that learns a shared tactile latent space across resistive, magnetic, and vision-based sensors. Using paired contact data for alignment, it enables zero-shot transfer of manipulation policies (pick-and-place, insertion, reorientation) trained with one sensor to a physically different one, improving average success rate from 27.5% to 45.9%.

- **[UniTac: A Unified Multimodal Model for Cross-Sensor Tactile Understanding and Generation](https://arxiv.org/abs/2606.31451)**
  Introduces UniTac, the first unified multimodal model for tactile understanding and generation across sensors. It models tactile interaction as a contact process with sensor and object attributes, achieving SOTA on tactile understanding tasks and generating realistic tactile signals.

- **[RCT: A Robot-Collected Touch-Vision-Language Dataset for Tactile Generalization](https://arxiv.org/abs/2606.31694)**
  Presents RCT, a large-scale dataset of 29,279 tactile frames from robot presses on 122 materials using DIGIT sensors. It exposes the challenge of novel-material generalization, where held-out material recall drops sharply, serving as a benchmark for generalizable tactile perception.

- **[RoboTacDex: A Dexterous Visual-Tactile-Action Dataset for Humanoid Manipulation](https://arxiv.org/abs/2606.31836)**
  Releases a large, multi-modal dataset from a Unitree G1 humanoid (6k trajectories, 19 tasks, 22 objects). Includes multi-view RGB-D, tactile feedback, and semantic annotations, focusing on dual-arm and dexterous hand tasks.

- **[UniTacVLA: Unified Tactile Understanding and Prediction in Vision Language Action Models](https://arxiv.org/abs/2606.31723)**
  Enhances VLA models with tactile reasoning for contact-rich manipulation. It models tactile signals as dynamic cues for both current state understanding and future contact prediction via a "tactile chain-of-thought," improving robustness in insertion, wiping, and assembly tasks.

- **[TacGen: Touch Is a Necessary Dimension of Physical-World Representation](https://arxiv.org/abs/2606.29173)**
  Argues that touch provides a necessary physical evidence channel for representation learning. Proposes TacGen, which aligns visuo-tactile representations and generates tactile latents from RGB, demonstrating significant gains in modeling contact-dependent properties like mass and force.

### Dexterous & Multi-Finger Grasping & Manipulation

- **[From Grasps to Dexterity: Large-Scale Grasp Pretraining for Dexterous Manipulation](https://arxiv.org/abs/2606.30749)**
  Shows that large-scale dexterous grasp datasets can be used to pretrain a low-level controller for functional dexterity. On the new DexCraft benchmark (6 articulated tool-use tasks), this approach outperforms end-to-end diffusion policies, with full-task success improved by 33.3 pp over DP3 in the real world.

- **[CoDex: Learning Compositional Dexterous Functional Manipulation without Demonstrations](https://arxiv.org/abs/2606.31909)**
  Introduces a zero-demonstration framework for dexterous functional object manipulation (e.g., aiming a spray bottle). Uses VLMs to infer semantic constraints that guide analytic optimization and RL to discover full grasp-move-actuate policies, demonstrating success with a 16-DoF hand on novel objects.

- **[GRIT: Learning Dexterous Grasping from Sparse Taxonomy Guidance](https://arxiv.org/abs/2604.04138)**
  Proposes a two-stage framework: first predicting a taxonomy-based grasp specification, then using it to condition a policy for generating finger motions. This sparse command approach improves generalization to novel objects (87.9% success) and allows user control by selecting grasp strategies.

### Visuomotor Policies & VLA Models

- **[ELASTIC: Efficiently Learning to Adaptively Scale Test-Time Compute for Generative Control Policies](https://arxiv.org/abs/2606.31132)**
  Learns state-dependent test-time compute schedules for diffusion policies and VLAs. By adaptively allocating sequential denoising steps and parallel samples, ELASTIC matches Best-of-10 success on a real VLA model while reducing wall-clock latency by 34%.

- **[MIRTH: Mutual-Information Reasoning with Temporal Hubs for Vision-Language-Action Agents](https://arxiv.org/abs/2606.31167)**
  Addresses VLA limitations with dual-scale temporal memory hubs, latent reasoning tokens optimized via mutual information, and parallel action decoding. Achieves SOTA on LIBERO and real-world LeRobot tasks, showing emergent error recovery.

- **[3D HAMSTER: Bridging Planning and Control in Hierarchical VLA Models through 3D Trajectory Guidance](https://arxiv.org/abs/2606.31329)**
  Solves the depth ambiguity in 2D trajectory-guided VLAs by having the planner predict metric 3D trajectories directly. Augmenting a VLM with a depth encoder, it outputs 3D waypoints for a pointcloud-based policy, consistently outperforming 2D-guided baselines.

- **[Z-1: Efficient Reinforcement Learning for Vision-Language-Action Models](https://arxiv.org/abs/2606.31846)**
  Presents an RL post-training framework for flow-based VLA models ($\pi_{0.5}$). Using task-wise GRPO on 24 RoboCasa tasks, Z-1 improves the average success rate from 67.4% (SFT) to 80.6%, outperforming published SOTA models.

- **[LARA: Latent Action Representation Alignment for Vision-Language-Action Models](https://arxiv.org/abs/2606.07100)**
  Proposes LARA, a plug-and-play framework that jointly optimizes a Latent Action Model and a VLA via representation alignment. This regularizes VLA learning with forward dynamics, yielding ~10% improvement on simulation and real-world benchmarks.

- **[RhinoVLA Technical Report](https://arxiv.org/abs/2606.07383)**
  A deployment-oriented VLA model co-designed with the Huixi R1 edge SoC. Achieves 11.69 Hz end-to-end inference on-chip by optimizing tokens, hardware-aware compilation, and a unified robot interface, performing comparably to $\pi_{0.5}$.

### Policy Learning & Reinforcement Learning

- **[Efficient Sim-to-Real Transfer of World-Action Models from Synthetic Priors](https://arxiv.org/abs/2606.31101)**
  Demonstrates the first successful sim-to-real transfer of a video diffusion-based world-action model for manipulation. Using ~800 synthetic demos per task and domain randomization, the policy attains a 35% average success rate zero-shot on a Franka robot.

- **[Stage-Transition Dense Reward Modeling for Reinforcement Learning](https://arxiv.org/abs/2606.31377)**
  Proposes STDR, a visual reward-learning framework that infers stage structure from expert videos to provide dense rewards for RL. It combines stage-transition and within-stage progress feedback, improving sample efficiency on 14 manipulation tasks.

- **[Revisiting Parameter Redundancy in Vision-Language-Action Models](https://arxiv.org/abs/2606.31382)**
  Analyzes parameter redundancy in VLAs post-adaptation. Finds that while VLAs are sensitive to pruning, a structured, multi-module pruning scheme can reduce parameters by 12-30% while maintaining ~90% performance without fine-tuning, outperforming existing criteria.

- **[Multisensory Continual Learning: Adapting Pretrained Visuomotor Policies to Force](https://arxiv.org/abs/2606.30988)**
  Studies adapting a pretrained vision-only policy to new tasks with force-torque sensing. Proposes MuSe, which uses multi-stage fusion and experience replay, showing that modest multisensory data can improve performance on contact-rich tasks without degrading original capabilities.

- **[Freeform Preference Learning for Robotic Manipulation](https://arxiv.org/abs/2606.32027)**
  Introduces FPL, which learns robot policies from freeform human preferences defined along natural-language axes (e.g., speed, safety). It trains a language-conditioned reward model and reward-conditioned policy, improving over sparse-reward methods by 38 pp on long-horizon tasks.

- **[Adapting Generalist Robot Policies with Semantic Reinforcement Learning](https://arxiv.org/abs/2606.31958)**
  Proposes Semantic Action RL (SARL), which treats language prompts as an action space to adapt generalist policies via RL. By modulating prompts to elicit existing skills, SARL unlocks new long-horizon capabilities and significantly outperforms direct action-optimization approaches.

- **[Warp RL: Reshaping Base Policy Distributions for Dynamics Adaptation](https://arxiv.org/abs/2606.31043)**
  Addresses the limitations of additive residual RL by proposing Warp RL, which applies an invertible, state-conditioned transformation (spline flow) to the base policy distribution. This allows reshaping of the action distribution, improving adaptation under dynamics shift.

- **[ChronoFlow-Policy: Unifying Past-Current-Future Interaction Flow in Visuomotor Policy Learning](https://arxiv.org/abs/2606.31493)**
  Introduces ChronoFlow, a temporally unified representation of past, current, and future interaction dynamics via 3D keypoints. The diffusion-based policy jointly learns ChronoFlow and actions, outperforming diffusion policy baselines on long-horizon and non-Markovian tasks.

- **[Wavelet Policy: Imitation Learning in the Scale Domain with World Prior Memory](https://arxiv.org/abs/2504.04991)**
  Proposes a lightweight imitation learning framework that uses World Prior Memory (from static images) and wavelet-based multi-scale action modeling. Outperforms strong baselines on simulated and real-world manipulation tasks.
