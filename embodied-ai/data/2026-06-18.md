Here is a summary of today's arXiv papers from robotics and AI feeds, grouped by topic.

### Dexterous Manipulation & Grasping
*   **[Do as I Do: Dexterous Manipulation Data from Everyday Human Videos](https://arxiv.org/abs/2606.19333)** - A method to reconstruct and retarget monocular RGB human videos (egocentric/exocentric) to multi-fingered dexterous robotic hands for generating real-world manipulation data.
*   **[Zero-Shot Long-Horizon Dexterous Manipulation via Multi-View 3D-Grounded VLM Reasoning](https://arxiv.org/abs/2606.19340)** - A zero-shot framework that uses a VLM on multi-view images to generate 3D task plans for dexterous pick-and-place and tool-use, combining 3D keypoints with task-conditioned grasp affordances.
*   **[EAGG: Embodiment-Aligned Grasp Generation via Geometry-Aware Graph Conditioning](https://arxiv.org/abs/2606.18092)** - Introduces a cross-end-effector grasp generator that uses a topology-aware end-effector graph and iterative geometry injection to generalize across parallel grippers and dexterous hands.
*   **[GCNGrasp-VP: Affordance-Guided View Planning for Efficient Task-Oriented Grasping](https://arxiv.org/abs/2606.19091)** - Integrates affordance field prediction with active view planning to improve task-oriented grasping under occlusion, outperforming scene-uncertainty-driven baselines.

### Visuomotor Policies & Imitation Learning
*   **[DREAM-Chunk: Reactive Action Chunking with Latent World Model](https://arxiv.org/abs/2606.18589)** - Augments action-chunking VLA policies with a lightweight latent world model for test-time scaling, improving robustness under stochastic dynamics.
*   **[Invertible Neural Network Adapter for One-Step Flow Matching in Robot Manipulation](https://arxiv.org/abs/2606.19194)** - A one-step denoising flow-matching adapter for VLA models that reduces inference latency (110ms to 61ms) while maintaining performance.
*   **[Object-Centric Residual RL for Zero-Shot Sim-to-Real VLA Enhancement](https://arxiv.org/abs/2606.18953)** - Learn a residual RL policy in simulation using object poses to refine VLA actions on a real Franka robot, improving success from 42% to 76% zero-shot.
*   **[Motion-Focused Latent Action Enables Cross-Embodiment VLA Training from Human EgoVideos](https://arxiv.org/abs/2606.18955)** - A VQ-VAE framework that decouples motion dynamics from backgrounds to learn a cross-embodiment action codebook from unlabeled human videos for VLA pre-training.
*   **[Learning to Annotate Delayed and False AEB Events](https://arxiv.org/abs/2606.19186)** - An automated annotation system for rare (delayed/false) Autonomous Emergency Braking events, tackling extreme class imbalance and asymmetric label noise.

### Tactile & Force Sensing
*   **[TactSpace: Learning a Physics-enriched Shared Latent Space for Tactile Sim-to-Real Transfer](https://arxiv.org/abs/2606.18959)** - A multi-modal representation learning framework that aligns simulated and real tactile data (e.g., penetration depth vs. capacitance) in a shared latent space for zero-shot transfer.
*   **[HT-Bench: Benchmarking and Learning Dexterous Full-Hand Tactile Representations](https://arxiv.org/abs/2606.19161)** - A large-scale benchmark (10M RGB, 7.8M tactile frames) for evaluating dexterous full-hand tactile representations, plus a VQ vision-tactile encoder (HandTouch) for fine-grained retrieval and synthesis.
*   **[Self-Supervised Mask-Aware Transformers for Fault-Tolerant FBG Force Sensing](https://arxiv.org/abs/2606.18628)** - A unified, self-supervised Transformer for Fiber Bragg Grating force sensors that handles intermittent channel dropouts, achieving graceful degradation under sensor failure.

### World Models, Simulation & Sim-to-Real
*   **[PAIWorld: A 3D-Consistent World Foundation Model for Robotic Manipulation](https://arxiv.org/abs/2606.18375)** - A diffusion-transformer world model that enforces multi-view 3D consistency using Geometry-Aware Cross-View Attention and Geometric RoPE.
*   **[WEAVER: An Effective World Model for Robotic Manipulation](https://arxiv.org/abs/2606.13672)** - A multi-view VLA world model trained with flow-matching loss for long-horizon manipulation, demonstrating strong policy evaluation, improvement, and test-time planning on real hardware.
*   **[Mem-World: Memory-Augmented Action-Conditioned World Models for Persistent Robot Manipulation](https://arxiv.org/abs/2606.18960)** - A world model with a 4D wrist-view-centered memory (W-VMem) that uses surfel indexing to retrieve relevant history for reliable long-horizon rollout generation.
*   **[SC3-Eval: Evaluating Robot Foundation Models via Self-Consistent Video Generation](https://arxiv.org/abs/2606.18610)** - A recipe for using a pre-trained video foundation model as a policy evaluator, enforcing forward-inverse dynamics and cross-view consistency.
*   **[Kairos: A Native World Model Stack for Physical AI](https://arxiv.org/abs/2606.16533)** - A comprehensive world model stack with a Native Unified Architecture using hybrid linear-temporal attention for long-horizon state propagation and error control.

### Teleoperation & Human-Robot Interaction
*   **[Seeing Through Occlusion: Deterministic Arm Kinematic Correction for Robot Teleoperation](https://arxiv.org/abs/2606.19240)** - A method to correct degraded depth estimates from a single RGB-D camera during teleoperation by enforcing geometric arm-length constraints.
*   **[Admittance-Based Surface Alignment for Human-in-the-Loop Robotic Visual Inspection](https://arxiv.org/abs/2606.18601)** - A real-time control pipeline that unifies operator input with perception-driven surface alignment using an admittance-based framework for compliant visual inspection.

### Humanoid & Legged Locomotion
*   **[HALOMI: Learning Humanoid Loco-Manipulation with Active Perception](https://arxiv.org/abs/2606.18772)** - A scalable framework combining egocentric human demonstration data with a manifold-constrained controller for whole-body head-hand tracking on a G1 humanoid.
*   **[Mobile Pedipulation for Object Sliding via Hierarchical Control on a Wheeled Bipedal Robot](https://arxiv.org/abs/2606.19233)** - A hierarchical NMPC framework enabling a wheeled bipedal robot to perform planar object sliding with its feet (pedipulation), demonstrated on a real robot.
*   **[SRL: Combining SLIP Model and Reinforcement Learning for Agile Robotic Jumping](https://arxiv.org/abs/2606.18625)** - A hybrid framework integrating SLIP model priors with RL-driven feedback for agile and stable jumping in bipedal and quadrupedal robots.

### Navigation & SLAM
*   **[VEGA: Learning Navigation VLAs from In-the-Wild Egocentric Video](https://arxiv.org/abs/2606.18426)** - Trains a navigation VLA from unlabeled egocentric videos by reconstructing scene geometry to generate obstacle-aware trajectories for training.
*   **[EffiNav: Fusing Depth and Vision-Language for Efficient Object Goal Navigation](https://arxiv.org/abs/2606.18634)** - An efficient object-goal navigation framework evaluated on HM3D and OVON, matching or outperforming baselines while being practical for physical robots.
*   **[FAST-LIVGO: A Degeneracy-Robust LiDAR-Inertial-Visual-GNSS Fusion Odometry](https://arxiv.org/abs/2606.19190)** - A tightly-coupled multi-sensor fusion odometry framework with degeneracy-aware dual-mode outlier rejection for robustness in harsh conditions.
*   **[A High-accuracy Event-based Underwater SLAM System](https://arxiv.org/abs/2606.18951)** - The first high-accuracy event-based underwater stereo SLAM system, using Bayesian Optimization and a "latest-observation-first" triangulation to overcome degraded imaging.

### Misc. Robotics & AI
*   **[Recover, Discover, Plan: Learning Skills and Concepts from Robot Failures](https://arxiv.org/abs/2606.18328)** - A framework (ReSYNC) that uses RL to learn recovery skills and then discovers relational predicates for abstract planning, enabling generalization to unseen problems.
*   **[Guava: An Effective and Universal Harness for Embodied Manipulation](https://arxiv.org/abs/2606.18363)** - A harness framework that distills manipulation capabilities into a 4B model using fewer than 2K trajectories, performing comparably to frontier models.
*   **[Qwen-RobotManip: Alignment Unlocks Scale for Robotic Manipulation Foundation Models](https://arxiv.org/abs/2606.17846)** - A generalizable VLA foundation model built with a unified alignment framework across representation, motion, and behavior, enabling emergent capabilities from a large (~38,100h) corpus.
*   **[First Proof Second Batch](https://arxiv.org/abs/2606.18119)** - A test of current AI systems on ten original research-level mathematics problems, all of which scored below 10%, highlighting a gap between competition and research-level reasoning.
