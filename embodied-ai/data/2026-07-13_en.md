### 1. Tactile Sensing and Dexterous Manipulation

- **[TactiDex: A Real-World Tactile-Guided Benchmark for Human-Like Dexterous Manipulation](https://arxiv.org/abs/2607.09190)**
    - Introduces a benchmark and framework (TactiSkill) for tactile-guided dexterous manipulation. It uses a tri-component tactile reward to align whole-hand tactile signals with kinematic states, enabling more human-like and physically grounded robot execution.
- **[Tactile and Vision Conditioned Contact-Centric Control for Whole-Arm Manipulation](https://arxiv.org/abs/2607.09218)**
    - Presents TACTIC, a receding-horizon controller for whole-arm manipulation. It uses a hybrid predictive model combining RGB-D, tactile, and proximity sensing with a sampling-based MPC to manage multi-contact trajectories during tasks like repositioning a manikin.
- **[SpikeATac: A Multimodal Tactile Finger with Taxelized Dynamic Sensing for Dexterous Manipulation](https://arxiv.org/abs/2510.27048)**
    - Introduces SpikeATac, a multimodal tactile finger combining dynamic PVDF and static capacitive sensing. This enables tasks like in-hand manipulation of fragile objects by providing fast slip and contact onset detection for force-modulating policies.
- **[Tactile Genesis: Exploring Tactile Sensors at Scale for Learning Dexterous Tasks](https://arxiv.org/abs/2606.22332)**
    - A GPU-parallel tactile simulator that supports multiple sensor types. Key findings for policy learning: sensor placement (whole-hand vs. fingertip) is more critical than sensor type or high resolution, and per-taxel force/torque is the most useful modality.

### 2. Vision-Language-Action (VLA) Models and Policy Learning

- **[AgenticFocus: Object-Preserving Mixed Reality Synthesis from Human FPV Video for Dexterous Humanoid Learning](https://arxiv.org/abs/2607.08857)**
    - A pipeline that converts human egocentric video into robot training data by restoring occluded objects, reconstructing hand motion, and retargeting to a humanoid, improving trajectory error and wrist smoothness.
- **[FlowDAgger: Human-in-the-Loop Adaptation of Generative Robot Policies in Latent Space](https://arxiv.org/abs/2607.08877)**
    - A method for adapting frozen flow-matching policies from human interventions by inverting expert actions into latent noise. This allows for rapid skill acquisition from few interventions while preserving pre-trained behavioral priors.
- **[CLAP: Direct VLM-to-VLA Adaptation via Language-Action Grounding](https://arxiv.org/abs/2607.08974)**
    - A method to convert VLMs into VLAs by prepending action sequences with natural-language descriptions, causally conditioning action prediction on a language plan without modifying the backbone. Achieves 90.8% on LIBERO with single-epoch fine-tuning.
- **[GenVid2Robot: From Video Generation to Robot Manipulation via Rigid-Geometric Consistency](https://arxiv.org/abs/2607.09191)**
    - A framework that converts generated video motion into executable robot trajectories by verifying geometric consistency against real RGB-D data, using sparse SE(3) models and grasp constraints.
- **[PhysV2A: Reachability-Gated and Semantic-Mask-Constrained Feasibility Completion for Video-to-Robot Manipulation](https://arxiv.org/abs/2607.09365)**
    - Converts video-derived object motion into executable robot trajectories by performing reachability checks on grasp-trajectory pairs and using semantic masks to refine trajectories via redundancy optimization.
- **[B-spline Policy: Accelerating Manipulation Policies via B-spline Action Representations](https://arxiv.org/abs/2607.09648)**
    - Proposes parameterizing robot actions as B-spline curves instead of discrete action chunks. This enables continuous, temporally scalable trajectories that can be executed at higher frequencies, reducing task completion time.
- **[FabriVLA: A Lightweight Vision-Language-Action Model for Precise Multi-Task Manipulation](https://arxiv.org/abs/2607.08575)**
    - A compact VLA (1B scale) combining a VLM backbone with a flow-matching action head. Achieves a 90.0% success rate on the MT50 benchmark, demonstrating strong performance without requiring massive parameters.

### 3. Real-to-Sim, Sim-to-Real, and Data Generation

- **[Re$^3$Sim: Generating High-Fidelity Simulation Data via 3D-Photorealistic Real-to-Sim for Robotic Manipulation](https://arxiv.org/abs/2502.08645)**
    - A real-to-sim system using 3D reconstruction and neural rendering to create photorealistic digital twins. Policies trained purely in simulation achieve zero-shot real-world transfer with over 58% success.
- **[PAC-ACT: Post-training Actor-Critic for Action Chunking Transformers](https://arxiv.org/abs/2607.09590)**
    - An RL post-training framework for ACT policies designed for contact-rich tasks. It reformulates optimization at the chunk level and uses a behavior-prior constraint to improve contact stability and force safety during online fine-tuning.
- **[DemoBridge: A Simulation-in-the-Loop Toolkit for Single-View Human Demonstration Retargeting](https://arxiv.org/abs/2607.09519)**
    - A toolkit that retargets single-view human hand demonstrations to robot arm trajectories using a collision-aware planner and a physics simulator in the loop to validate and adjust the plan, ensuring executability.

### 4. Teleoperation and Shared Control

- **[Robot Trajectron V3: A Probabilistic Shared Control Framework for SE(3) Manipulation](https://arxiv.org/abs/2607.09315)**
    - A shared control framework for high-DoF manipulation that formulates control as Bayesian inference. It learns a prior over user intent using a transformer and combines it with real-time commands to reduce user workload and improve success rate.
- **[WAM-TTT: Steering World-Action Models by Watching Human Play at Test Time](https://arxiv.org/abs/2607.06988)**
    - A test-time training framework that steers a frozen world-action model by absorbing raw human videos into a lightweight adaptive memory via self-supervised prediction, enabling task steering without robot actions or fine-tuning.

### 5. 3D Scene Representation and Control

- **[SplatCtrl: Perception-Action Coupling via Gaussian Scene Representations and Reactive Robot Control](https://arxiv.org/abs/2607.08948)**
    - A framework for real-time 3D scene reconstruction and reactive control. It derives continuous distance functions from 3D Gaussians for collision probability estimates in control barrier functions, enabling collision-free robot arm control in dynamic environments.
