### Dexterous Manipulation & Hands

* **[PDS Joint: A Parametric Double-Spiral Joint Tailored for Dexterous Hands](https://arxiv.org/abs/2606.24377)** - Presents a parametric double-spiral (PDS) compliant joint for dexterous hands, enabling systematic shaping of directional stiffness. Includes co-designed proprioception and a learning-based calibration pipeline. Integrated into an open-source hand that demonstrates safe human interaction and object grasping.
* **[TouchThinker: Scaling Tactile Commonsense Reasoning to the Open World...](https://arxiv.org/abs/2606.11637)** - Proposes TouchThinker, a framework to scale tactile commonsense reasoning. Introduces a million-scale, multi-source tactile reasoning dataset (TouchThinker-1M) and an action-aware modeling mechanism to improve tactile representation efficiency.
* **[Skill Shadowing Degrades Performance When Expanding Skill Libraries](https://arxiv.org/abs/2605.24050)** - Investigates the performance degradation of LLM agents as their skill libraries grow. Formulates the problem and identifies "skill shadowing" (selecting wrong skills) as the primary bottleneck, significantly more important than context overhead.

### Tactile Sensing & In-Hand Manipulation

* **[TACTFUL: Tactile-Driven Exploration For Object Localization and Identification in Confined Environments](https://arxiv.org/abs/2606.24712)** - A vision-free tactile exploration framework that uses a multi-fingered robot to autonomously explore confined spaces, discover objects, and identify them via tactile reconstruction. Trained entirely on real hardware.
* **[NoContactNoWorries: Estimating Contact through Vision and Proprioception for In-Hand Dexterous Manipulation](https://arxiv.org/abs/2606.24450)** - A transformer-based framework that infers binary contact states from RGB-D vision and proprioception, acting as a pseudo-tactile signal for in-hand object reorientation. Validated in both simulation and real-world experiments.
* **[MILE: A Mechanically Isomorphic Hand Exoskeleton and Visuotactile Robotic Hand...](https://arxiv.org/abs/2512.00324)** - A teleoperation system with a mechanically isomorphic exoskeleton and robotic hand for collecting synchronized multimodal demonstrations, including visuotactile data. Reduces reliance on IK-based retargeting.
* **[TactileReflex: Noise-Statistics-Driven Vision-Tactile Reflex Control...](https://arxiv.org/abs/2605.23568)** - A reflex control paradigm using vision-based tactile sensing. Derives control thresholds from sensor noise statistics to enable force-sensitive manipulation of fragile objects (e.g., holding a plastic cup of liquid). Demonstrates high success rates in dynamic pouring.

### Visuomotor Policies & Imitation Learning

* **[MinInter: Minimizing Trajectory Interpolation During Data Augmentation for Imitation Learning](https://arxiv.org/abs/2606.24078)** - Introduces MinInter, a method for trajectory-level data augmentation that selects the source demonstration requiring the least interpolation for a given initial state. Improves data quality and policy success rates on 12 manipulation tasks, especially for contact-rich and long-horizon tasks.
* **[RE4: Transformation-aware Imitation of Object Interactions Using Manipulation Modes](https://arxiv.org/abs/2606.24403)** - An imitation learning framework (RE4) that uses object-centric pose estimation and manipulation mode-aware retrieval to create interpretable and robust manipulation skills. Outperforms end-to-end methods in low-data regimes.
* **[Beyond Monotonic Progress: Retry-Supervised Value Learning for Robot Imitation](https://arxiv.org/abs/2606.24633)** - A framework (ReTVL) for learning fine-grained, mistake-sensitive value functions from imperfect demonstrations by leveraging retry events as sparse supervision. Improves imitation learning from mixed-quality data.
* **[Geometric Action Model for Robot Policy Learning](https://arxiv.org/abs/2606.17046)** - A language-conditioned manipulation policy (GAM) that repurposes a pretrained geometric foundation model (GFM) as a shared backbone for perception, temporal prediction, and action decoding. Outperforms current VLA baselines on simulation and real-robot benchmarks.
* **[G$^3$VLA: Geometric inductive bias for Vision-Language-Action Models](https://arxiv.org/abs/2606.24472)** - Injects calibrated geometry into the visual-token stream of a pretrained VLA via ray embeddings, projective positional encoding, and cross-view fusion. Yields consistent gains on LIBERO, RoboCasa, and real-robot tasks.
* **[Origami Policy: Learning Parquet Assembly from Human Demonstrations](https://arxiv.org/abs/2606.23768)** - (Title inferred from context, likely paper [Supervise What Survives: Geometry-Guided VLA Adaptation from Synthetic Robot Videos](https://arxiv.org/abs/2606.24448)) - Proposes GRA, which extracts the geometric content (future 2D waypoints) from generated robot videos to supervise visual perception, while training the action head only on real demonstrations. Outperforms pseudo-action baselines.

### Sim-to-Real & World Models

* **[Sim-to-Real Betting on the E-Process...](https://arxiv.org/abs/2606.24038)** - Integrates sim-to-real performance estimates with "betting" techniques to create reliable, anytime-valid confidence sequences for mean estimates, particularly useful in robot performance testing.
* **[World Value Models for Robotic Manipulation](https://arxiv.org/abs/2606.24742)** - Introduces the World Value Model (WVM), which marries world models with value estimation to provide accurate task progressions for assessing data quality. Demonstrates state-of-the-art performance on Value-Order Correlation and improves policy learning from mixed-quality data.
* **[PAIWorld: A 3D-Consistent World Foundation Model for Robotic Manipulation](https://arxiv.org/abs/2606.18375)** - A world foundation model that achieves multi-view 3D consistency in robotic manipulation via geometry-aware cross-view attention and 3D feature distillation. Enables model-based planning and world action models.
* **[Grounding Generative Policies in Physics: Optimization-Guided Diffusion for Robot Control](https://arxiv.org/abs/2606.24208)** - An inference-time optimization framework that couples diffusion-based behavior generation to physical feasibility via constrained optimization guidance. Improves task success on dexterous grasping and visuomotor manipulation.
* **[DynaWM: Dynamics-Aware Distillation...for Smooth Locomotion over Continuous Stairs](https://arxiv.org/abs/2606.24089)** - A framework for bipedal-wheeled robot locomotion that uses a world model as a regularizer to enforce forward-dynamics awareness. Enables smooth traversal of continuous stairs by enhancing terrain encoding.
* **[Dynamics, stability, and energy efficiency of an energy-recycling rimless wheel](https://arxiv.org/abs/2606.22073)** - Proposes a passive walking rimless wheel with a lockable spring-clutch mechanism that stores and re-injects impact energy, reducing the Cost of Transport by up to 16%.
