### 具身智能与灵巧操作

1.  **[Learning Dexterous Manipulation Using Contact Wrench Guidance From Human Demonstration](https://arxiv.org/abs/2607.00033)** (CHORD)
    *   提出了 CHORD 框架，通过物体中心接触力/力矩空间引导，利用强化学习从人类演示中学习灵巧操作技能。该框架在包含 4739 个任务的大规模仿真基准上平均成功率达 82.12%，并成功迁移到真实机器人上。
2.  **[Human-Centric Transferable Tactile Pre-Training for Dexterous Robotic Manipulation](https://arxiv.org/abs/2607.01067)** (H-Tac & TTP)
    *   为解决触觉数据稀缺问题，提出了 H-Tac 大规模触觉-动作数据集（160小时人类视频），并在此基础上开发了可迁移的触觉预训练（TTP）系统。通过统一人类和机器人的触觉与动作空间，TTP 在仿真和真实机器人上展现出强大的泛化能力和精细操作能力。
3.  **[Neural Surface and Reflectance Modelling from 3D Radar Data](https://arxiv.org/abs/2603.25623)**
    *   提出了一种神经隐式方法，用于从雷达点云中进行 3D 建图。该方法联合建模场景几何和视角相关的雷达强度，在低可见度环境下优于基于 LiDAR 的重建方法，为机器人在恶劣环境下的感知提供了新思路。
4.  **[Learning Gait-Aware Quadruped Locomotion with Temporal Logic Specifications](https://arxiv.org/abs/2607.00442)**
    *   引入信号时序逻辑来为四足机器人的不同步态（如行走、小跑）指定参数化约束。该方法将 STL 鲁棒性作为奖励，通过 PPO 学习出更稳定、速度跟踪更精准的步态策略。

### 视觉-运动策略与操作模型

5.  **[Unleashing More Actions via Action Compositional Training for VLA Models](https://arxiv.org/abs/2607.00351)** (ACT-VLA)
    *   针对 VLA 模型在分布外场景中泛化能力差的问题，提出了 ACT-VLA，一种无需额外真实数据的离线数据增强框架。它利用模型自身的潜任务表征，从现有任务中合成新的、物理上有效的演示，扩展了训练分布。
6.  **[Domain Arithmetic: One-Shot VLA Adaptation under Environmental Shifts](https://arxiv.org/abs/2607.00666)** (DART)
    *   为解决 VLA 模型在环境变化（如相机位姿、机器人本体改变）下的适应问题，提出了 DART，一种基于类比的方法。它仅需收集一次演示，通过权重向量算术进行域适应，在仿真和真实实验中效果显著。
7.  **[FurnitureVLA: Learning Long-Horizon Bimanual Furniture Assembly with Vision-Language-Action Model](https://arxiv.org/abs/2607.01212)**
    *   首次系统性地研究了真实尺度双臂家具组装任务。提出了一个进展增强的VLA模型，通过联合预测动作和连续的“进展信号”来自动实现子任务切换，减少了长时域任务的累积误差。
8.  **[Structured 4D Latent Predictive Model for Robot Planning](https://arxiv.org/abs/2607.01166)**
    *   提出了一种结构化的4D潜空间预测模型，用于机器人规划。该模型在结构化潜空间内预测场景3D结构的演化，解码后可生成多种3D格式，显著优于基于2D视频的规划器，具备更好的3D一致性和多视角连贯性。
9.  **[3D Point World Models: Point Completion Enables More Accurate Dynamics Learning](https://arxiv.org/abs/2607.00148)** (3DPWM)
    *   提出了3D点云世界模型，通过先补全部分点云，再学习动作条件下的动力学。该方法在长时域推演中表现可靠（100-300+步），并支持模型预测控制的规划，成功实现了 sim-to-real 迁移。
10. **[Joint Discovery of Object and Action Symbols through Effect Prediction for Robotic Manipulation Planning](https://arxiv.org/abs/2607.00031)**
    *   提出了一种通过预测交互效果来联合发现高维操作原语和物体类别的方法。通过二值瓶颈层实现，并利用离散规划方法完成如堆叠等表上操作任务。

### 触觉感知与物理属性估计

11. **[PhyPush: One Push is All You Need for Sensorless Physical Property Estimation with Physics-Guided Transformers](https://arxiv.org/abs/2605.26284)**
    *   提出 PhyPush，一种仅通过单次推动的末端执行器速度即可估计物体质量和摩擦系数的物理引导 Transformer。它无需力/扭矩传感器，通过物理引导损失提升了物理一致性和泛化能力。
12. **[Wake up for Touch! Mask-isolated Tactile Alignment Learning in MLLMs](https://arxiv.org/abs/2607.00302)** (Splash)
    *   针对多模态大模型整合触觉时面临的“零和”权衡问题，提出了 Splash，一种掩码隔离的触觉对齐学习框架。它将参数空间划分为休眠和关键子空间，在守护通用视觉知识的同时，无损地扩展了触觉感知能力。

### 仿真到现实迁移与仿真环境

13. **[Physics Models for Sim-to-Real Transfer in Professional-Level Robot Table Tennis](https://arxiv.org/abs/2606.28805)**
    *   针对高水平机器人乒乓球中的 sim-to-real 挑战，提出了精确的空气动力学、球桌接触和球拍接触的物理模型。这些模型在涵盖职业比赛的大规模数据集上显著降低了预测误差，使得训练的策略首次能够与职业选手对战。
14. **[Warp RL: Reshaping Base Policy Distributions for Dynamics Adaptation](https://arxiv.org/abs/2606.31043)**
    *   提出 Warp RL，一种通过可逆变换重塑基策略动作分布来适应动力学变化的方法。它优于传统的残差强化学习，在仿真和真实机器人的 peg-insertion 任务中均表现更佳。
15. **[Building a Scalable, Reproducible, Evaluatable, and Closed-Loop Simulation Environment Foundation for Embodied Intelligence](https://arxiv.org/abs/2606.27962)**
    *   提出了一套云原生的仿真基础设施，支持大规模训练、标准化评估和数据采集。它集成了D-VLA, RL-VLA等系统，通过容器化、弹性调度等技术，为具身智能的发展提供统一平台。

### 遥操作与人机交互

16. **[Visualizing Impedance Control in Augmented Reality for Teleoperation: Design and User Evaluation](https://arxiv.org/abs/2603.25418)**
    *   提出了一种增强现实可视化方案，将阻抗控制器的目标位姿和偏移量直观地展示给操作者，以改善低力觉反馈的遥操作体验。用户实验表明，该方案在力关键型任务（如抬举）中减少了24%的完成时间。
17. **[Sensorless Four-Channel Control Architecture Using Inverse Dynamics Modeling for Human-Scale Bilateral Teleoperation](https://arxiv.org/abs/2607.01201)**
    *   提出了一种基于逆动力学建模的无传感器四通道遥操作架构，无需昂贵且易受噪声影响的力/扭矩传感器。在 WAM 机器人平台上验证了其在位置和力跟踪方面的优越性。

### 其他相关

18. **[EmbodimentSemantic: A Spatial Scene-Graph Dataset and Benchmark for Vision-Language Models on Embodied Manipulation Trajectories](https://arxiv.org/abs/2607.00020)**
    *   提出了 EmbodimentSemantic，一个用于评估 VLA 系统中空间关系理解的场景图数据集和基准。研究发现现有模型在精确的深度感知和视角依赖的空间结构理解上仍有困难。
19. **[Learning from Demonstration via Spatiotemporal Tubes for Unknown Euler-Lagrange Systems](https://arxiv.org/abs/2607.00534)** (STT-LfD)
    *   提出了 STT-LfD 框架，将演示视为数据驱动的安全规范（时空管）。利用异方差高斯过程学习任务的时间精度要求，并通过闭环反馈控制器执行，避免了系统辨识。
20. **[ABot-M0.5: Unified Mobility-and-Manipulation World Action Model](https://arxiv.org/abs/2607.00678)**
    *   提出了 ABot-M0.5，一种面向移动操作的世界行动模型。通过时间粒度、动作空间和训练-测试一致性三个层面的对齐设计，在长时域任务和精细控制任务上取得了先进性能。
