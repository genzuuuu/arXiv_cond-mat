### 一、 视觉-运动策略、世界模型与 Sim-to-Real

1.  **[PhysCoRe: Physics-Corrected Residual World Models for Material-Aware Deformable Dynamics](https://arxiv.org/abs/2607.20653)**
    *   提出 PhysCoRe，一种物理校正残差世界模型。它结合可微分的物质点法(MPM)模拟器和两个前馈神经网络：一个从视觉观察推断每粒子弹性参数，另一个学习并校正模拟器内部动力学偏差。该方法支持在线材料识别，并在预测精度上优于现有基线。

2.  **[FORGE-plus: Force-Budgeted Recovery for Contact-Rich Assembly with a Frozen LLM Supervisor](https://arxiv.org/abs/2607.21227)**
    *   提出一个双层框架，利用“冻结”的大语言模型(LLM)作为主管，为精密装配任务分配力上限并从力反馈信号中选择恢复策略。底层强化学习(RL)控制器在力约束下执行。在脆性物体放置和齿轮插入任务中，该框架在无破坏的情况下成功完成所有评估回合，证明了LLM在接触任务中的规划潜力。

3.  **[GuidedAttention: Interpretable and Correctable Visual Attention for OOD-Robust Robot Manipulation via Imitation Learning](https://arxiv.org/abs/2607.21049)**
    *   提出 GuidedAttention，一种模仿学习框架，引入可解释和可校正的视觉注意力作为显式中间表示。用户可以在 rollout 初始化时检查和校正模型预测的注意力关键点，并由跟踪模块自动传播。该方法在仿真和真实实验中，特别是在分布外条件下，显著提升了机器人的操作性能。

4.  **[EgoRecovery: Acquiring Failure Recovery Ability Through Human Recovery Demonstration](https://arxiv.org/abs/2607.19745)**
    *   针对机器人难以收集失败恢复数据的问题，提出利用人类的第一人称视频作为可扩展的替代方案。EgoRecovery 是一个协同训练框架，将人类的恢复演示对齐到与机器人数据共享的紧凑“修正意图”空间中，只需少量机器人演示数据即可学习可执行的恢复行为，显著提高了从失败状态恢复的成功率。

5.  **[Robots Acquire Manipulation Skills in Seconds from a Single Human Video](https://arxiv.org/abs/2607.20033)**
    *   提出 HOST 框架，使机器人能从单个人类视频中，在数十秒内习得新技能。该方法通过一个自我预测的级联过程，将视频演示和机器人轨迹映射到共享的任务进度空间，然后机器人主动跟随演示过程并适应自身形态，保留了先前掌握的技能，大幅提升了技能获取效率。

6.  **[VPWEM: Non-Markovian Visuomotor Policy with Working and Episodic Memory](https://arxiv.org/abs/2603.04910)**
    *   提出 VPWEM，一种配备工作记忆和情景记忆的非马尔可夫视觉运动策略。它利用滑动窗口作为短期工作记忆，并引入一个基于 Transformer 的压缩器，将窗口外的历史观察递归压缩为固定的情景记忆嵌入，以几乎恒定的计算和存储开销利用长期信息，在记忆密集型任务上优于基线。

7.  **[TER-DAgger: Force-Aware Residual DAgger via Trajectory Editing for Precision Insertion with Impedance Control](https://arxiv.org/abs/2603.04038)**
    *   提出 TER-DAgger，一种力感知的人机回环模仿学习框架。它通过基于优化的轨迹编辑平滑融合策略 rollout 和人类修正轨迹，缓解了协变量偏移。同时引入力感知的故障预测机制，仅在预测力值与测量值出现偏差时才触发人类干预，在笛卡尔阻抗控制框架下执行策略，提高了精密插入任务的成功率。

8.  **[Scale Up Strategically: Learning Compositional Generalization via Bias-Aware Evaluation and Data Collection for Robotic Manipulation](https://arxiv.org/abs/2607.21582)**
    *   提出一个诊断框架来量化策略对指令中不同语义成分（如颜色、动词、物体）的“偏见”。研究发现策略倾向于依赖突出特征（如颜色），而忽略其他成分（如动词、大小）。基于此，提出一种偏见感知的数据收集策略，通过重新分配固定预算来覆盖被忽略的成分，能以更少的演示数据实现更好的泛化。

9.  **[FELT: Generating Tactile Signals from Vision for Visuo-Tactile Manipulation](https://arxiv.org/abs/2607.20683)**
    *   提出 FELT 框架，能从 RGB 观测数据合成每根手指的触觉压力图像。它使用大型预训练视觉编码器和轻量级查询解码器，在单一前馈过程中预测触觉信号。实验证明，无论是使用生成的触觉图像还是潜在触觉特征，都能提升策略在接触密集型任务中的成功率，减少了训练鲁棒视觉-触觉策略对真实触觉数据的依赖。

### 二、 灵巧操作、双手机器人与抓取

1.  **[BiCompoDiff: Grasp, Handover, Rotate: Bimanual Object Reorientation via Compositional Diffusion and Energy-Based Optimization](https://arxiv.org/abs/2607.21341)**
    *   提出 BiCompoDiff，一个组合扩散和基于能量的框架，用于双手机器人的物体重定向。它联合优化了抓取、交接、重抓取和运动规划。通过将预训练的抓取扩散模型与双手机器人规划的能量基模型结合，在逆向扩散过程中通过梯度指导来避免碰撞、确保轨迹平滑和交接可行性，相比基线方法，成功率和轨迹平滑度均有显著提升。

2.  **[URF: A Unified Robot Control-Policy Framework for Stable Contact Aware Manipulation](https://arxiv.org/abs/2607.20912)**
    *   针对学习型操作策略与底层控制器分离导致的刚性接触不稳定问题，提出统一控制策略框架(URF)。它使策略不仅能预测动作（虚拟目标、刚度矩阵），还能预测阻抗-导纳切换比率，从而根据接触情况动态调整控制器行为。该方法在翻箱和按压任务中减少了力冲击和工具损坏等故障。

### 四、 数据集、基准与平台

1.  **[TableVerse: A Large-scale Tabletop Dataset with Real-world Grounded Layouts for Generalizable Manipulation](https://arxiv.org/abs/2607.21017)**
    *   提出 TableVerse，一个全自动的 Real2Sim 管线，能从无结构的网络图像确定性重建生成高保真、仿真就绪的桌面场景。它自动生成任务条件化的无碰撞抓取和放置演示序列。构建的 TableVerse-100K 数据集包含 10万个物理一致的场景及交互轨迹，为通用机器人操作研究提供了可扩展、高质量的数据基础。

2.  **[AXIS: A Growable Community-Driven Data Engine for Scalable Robot Manipulation](https://arxiv.org/abs/2607.21588)**
    *   介绍 AXIS，一个可增长的、社区驱动的数据引擎和基准。它支持基于浏览器的遥操作进行大规模演示收集，并自动生成和验证新任务。数据集包含 207 个任务和 5 万条轨迹。在 AXIS 上持续预训练显著提升了 VLA 策略的总体成功率，并表现出与数据量增加一致的扩展性。

3.  **[i2Nav-Robot: A Large-Scale Indoor-Outdoor Robot Dataset for Multi-Sensor Fusion Navigation](https://arxiv.org/abs/2508.11485)**
    *   提出 i2Nav-Robot，一个用于室内外环境多传感器融合导航的大规模数据集。数据集集成了固态激光雷达、毫米波雷达、立体相机、IMU、GNSS 和轮式里程计等传感器，并提供了厘米级别的高质量真值，覆盖了超过 17公里 的多样化场景。

4.  **[TransBiolab: A Real-World Multi-View Dataset of Cluttered Transparent Biomedical Objects](https://arxiv.org/abs/2607.21071)**
    *   提出 TransBiolab，一个真实世界的 RGB-D 数据集，专注于杂乱场景中透明生物医学物体的操作。包含超过 16万帧多视角数据，提供 6D 姿态、分割掩码和深度信息，用于评估在复杂实验室环境下的物体识别和操作能力。

5.  **[Topology Inference in Distribution Systems](https://arxiv.org/abs/2607.20480)** (与机器人无直接关系)

6.  **[ZONDA: Zero-shot Object Navigation with Dynamic Avoidance in Multi-floor Environments](https://arxiv.org/abs/2607.21025)**
    *   提出 ZONDA，一种零样本目标导航框架，专门应对多楼层和动态行人场景。它整合了启发式的多楼层规划、多视角目标验证和动态行人避让模块，在真实双足机器人和仿真环境中验证了其鲁棒性。

7.  **[Robostral Navigate](https://arxiv.org/abs/2607.20785)**
    *   介绍 Robostral Navigate，一个 8B 参数的视觉语言模型，用于视觉导航。它仅使用单目 RGB 图像流，通过在图像空间中预测路标点来生成策略，无需深度传感器或预建地图，可在不同形态的机器人上零样本部署。在 R2R-CE 和 RxR-CE 基准上达到了新的最优水平。

### 五、 其他相关工作

1.  **[SevDiff: Severity-Conditioned Diffusion for Long-Tail Conflict Trajectory Generation](https://arxiv.org/abs/2607.20549)**
    *   提出 SevDiff，一个严重程度条件化的去噪扩散概率模型，用于生成特定冲突剧烈程度（以碰撞时间TTC衡量）的车辆交互轨迹，可用于自动驾驶辅助系统的测试评估。

2.  **[A real-time RGB-D perception pipeline for autonomous impact hammers in mining](https://arxiv.org/abs/2607.20748)**
    *   为采矿领域的自主液压冲击锤设计了一个实时 RGB-D 感知流水线，可同时生成操作上可行的岩石破碎姿态和无机器人工作空间的 3D 表示。

3.  **[Decentralized UAV Swarms for Ground Target Protection in GPS- and Communication-Denied Environments](https://arxiv.org/abs/2607.20710)**
    *   提出一种在无 GPS 和通信的环境中，使用自主无人机蜂群进行地面目标防御的流水线。无人机仅依靠机载传感器进行目标跟踪和编队协调，采用分散式包围技术来拦截敌对无人机。

4.  **[A Human-Inspired Framework for Robotic Craniotomy](https://arxiv.org/abs/2607.21058)** & **[RL-MACRO: A Cybernetic Closed-Loop Intelligence Framework for Multimodal Adaptive Robotic Craniotomy](https://arxiv.org/abs/2607.21113)**
    *   两篇论文关注机器人辅助开颅手术。第一篇（RL-MACRO）提出一个控制论闭环框架，通过融合力和声音信号来感知隐藏的温度状态，并利用强化学习动态调整切削参数，实现安全高效的自主骨切割。
