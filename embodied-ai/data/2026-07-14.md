### Dexterous Manipulation & Robotic Hands

**[EgoSteer: A Full-Stack System Towards Steerable Dexterous Manipulation from Egocentric Videos](https://arxiv.org/abs/2607.09701)**
Introduces a full-stack system for pre-training dexterous VLA models from 9.6K hours of egocentric video, enabling data-efficient real-robot post-training. The model, EgoSteer, demonstrates language-guided dexterity across 40+ tasks and few-shot adaptation to long-horizon tasks like box folding.

**[GraspGraphNet: Graph-Structured Multi-Embodiment Dexterous Grasp Generation](https://arxiv.org/abs/2607.11031)**
A topology-aware grasp generation framework representing robot hands as URDF-derived kinematic graphs. It uses conditional flow matching to directly generate palm poses and joint configurations, achieving 83.48% success across three different dexterous hands without retargeting or post-processing.

**[Towards Human-level Dexterous Teleoperation](https://arxiv.org/abs/2607.11481)**
Introduces TeleDexter, a hand-object co-tracking controller for human-level dexterous teleoperation. It maps operator intent into learned low-level contact execution via RL, achieving a 75% average success rate on challenging tasks like in-hand reorientation and tool use.

**[A Minimalist Retargeting-Guided Reinforcement Learning Recipe for Dexterous Manipulation](https://arxiv.org/abs/2607.11874)**
Presents REGRIND, a retargeting-guided RL pipeline that learns dexterous manipulation policies from a single human demo. Demonstrates object-centric keypoint tracking across two multi-fingered hands for tasks like scissor operation and screwdriver turning, with zero-shot sim-to-real transfer.

**[TactileReflex: Noise-Statistics-Driven Vision-Tactile Reflex Control for Force-Sensitive Manipulation](https://arxiv.org/abs/2605.23568)**
Proposes a tactile reflex control paradigm for force-sensitive manipulation (e.g., fragile cups). Uses noise-statistics-driven thresholds from vision-based tactile sensors to achieve slip suppression and force protection.

**[Learning Tactile-Aware Quadrupedal Loco-Manipulation Policies](https://arxiv.org/abs/2604.27224)**
Presents a tactile-aware loco-manipulation policy learning pipeline. Uses human demos to train a high-level policy that predicts end-effector trajectories and tactile interaction cues, combined with whole-body RL for zero-shot real-world deployment.

### Visuomotor Policies & Imitation Learning

**[TS-Mask VLA: 2D Temporal-Spatial Masking for Vision-Language-Action Model](https://arxiv.org/abs/2607.09818)**
Introduces a VLA framework using a discrete diffusion action expert with a 2D temporal-spatial masking strategy. Achieves 95.7% average success on LIBERO and state-of-the-art long-horizon performance on CALVIN.

**[Mixture of Frames Policy: Multi-Frame Action Denoising for Bimanual Mobile Manipulation](https://arxiv.org/abs/2607.11884)**
Proposes a diffusion policy that performs synchronized action denoising across multiple coordinate frames for bimanual mobile manipulation. Introduces a column-based 6D rotation representation for exact frame transformations in SE(3).

**[Action Map Policy: Learning 3D Closed-loop Manipulation via Pixel Classification](https://arxiv.org/abs/2607.10706)**
Casts 3D manipulation policy learning as a pixel classification problem in image space. Projects 3D actions onto camera planes, treating each pixel location as a discrete class to control dimensionality while retaining multi-modality.

**[A Single Diffusion-Policy Controller for Multi-Task Block Pushing](https://arxiv.org/abs/2607.10892)**
Explores training a single diffusion policy from scratch via RL for multi-task block pushing. Combines reverse curriculum generation with objective-centric representations for zero-shot sim-to-real transfer.

**[SegDiff: Segmented Trajectory Diffusion for Consistent and Adaptive Robot Manipulation](https://arxiv.org/abs/2607.11027)**
Integrates keypose-based and continuous prediction by decomposing demos into motion segments between keyposes. Uses a Dynamic Temporal Ensembling mechanism for long-horizon prediction with real-time refinement.

**[Pix2Act: Image-Space Manipulation Policies with Equivariant Augmentation](https://arxiv.org/abs/2607.11167)**
Generates continuous 2D keypoint trajectories in camera planes and recovers end-effector poses via triangulation. Analysis of symmetry properties enables equivariant augmentation for improved generalization.

**[From Non-Rigid to Rigid: ... for Multi-object Search in Unknown Environments](https://arxiv.org/abs/2607.10437)**
Proposes Inter-POMDP, an interleaved POMDP planning algorithm that decomposes multi-object search. Uses LLM-informed beliefs for high-level reasoning and obstacle-aware particle beliefs for low-level motion planning.

### Tactile Sensing & Contact-Rich Manipulation

**[PinFT: Miniature 5-Axis Force/Torque Sensor Embeddable to Tweezer-like Tool](https://arxiv.org/abs/2607.10000)**
Presents a miniature 5-axis capacitive F/T sensor designed for integration into tweezer-like tools. Enables fine manipulation tasks like grasping a sub-mm SMD capacitor and distinguishing success from failure via gradient-based force signatures.

**[TAC-LOCO: Unified Whole-Body Control for Quadrupedal TACtile-Informed LOCO-Manipulation](https://arxiv.org/abs/2607.10132)**
A tactile-augmented RL framework for legged robots. Encodes tactile array observations from compliant grippers into a latent representation for whole-body control, achieving a 47% reduction in grasping force with a minimal object drop rate.

**[GelNeuro: A Sensing-Computing Integrated Neuromorphic Tactile System](https://arxiv.org/abs/2607.05241)**
A fully integrated neuromorphic visuo-tactile system. Pairs a GelSight Mini with a neuromorphic SoC, using a spiking CNN for 15-class texture recognition with 96.3% accuracy and extremely low power consumption (19.6 mW).

**[TACTIC: Tactile and Vision Conditioned Contact-Centric Control for Whole-Arm Manipulation](https://arxiv.org/abs/2607.09218)**
A receding-horizon controller for whole-arm manipulation combining RGB-D, distributed tactile sensing, and a 2D proximity representation. Uses a hybrid predictive model coupling learned dynamics with analytical kinematics via contact Jacobians.

**[D-SafeMPC: Diffusion-Driven Safe Model Predictive Control with Discrete-Time Control Barrier Functions](https://arxiv.org/abs/2607.10842)**
Enhances interaction between diffusion models and MPC by guiding the reverse diffusion process with CBFs/CLFs. Demonstrates improved safety and task success for a Franka manipulator in both sim and sim-to-real experiments.

### Sim-to-Real & Robot Learning

**[Robo-ValueRL: Reliable Value Estimation for Offline-to-Online Reinforcement Learning](https://arxiv.org/abs/2607.09866)**
A framework for reliable value estimation in offline-to-online RL. Learns a history-conditioned value estimator and uses its reliability to guide policy pretraining and online improvement. Achieves 86% on chip insertion and 84% on block disassembly.

**[Diffusion for Long-Horizon Multi-Robot Path Planning in Human-Shared Environments](https://arxiv.org/abs/2607.09911)**
Proposes Multi-Robot Rolling Diffusion (MRRD) for real-time path planning in dense crowds. Combines rolling-horizon planning, parallelized diffusion inference, and conflict-based search, scaling to 15 robots in real-time.

**[SUREFlow: State-space Uncertainty-aware REsidual Flow Matching for Robust Robot Manipulation](https://arxiv.org/abs/2607.10504)**
A state-space uncertainty-aware residual flow matching framework using a Mamba backbone. Jointly predicts action velocities and input-dependent uncertainty, achieving 92.5% SR on LIBERO with only 179M parameters.

**[ReMoSPLAT: Reactive Mobile Manipulation Control on a Gaussian Splat](https://arxiv.org/abs/2512.09656)**
A reactive controller for mobile manipulation based on quadratic programming that leverages a Gaussian Splat representation for collision avoidance. Validated on both simulation and real robot platforms.

**[TAC-LOCO: Unified Whole-Body Control for Quadrupedal TACtile-Informed LOCO-Manipulation](https://arxiv.org/abs/2607.10132)**
Already covered under "Tactile Sensing". Purely tactile-augmented whole-body control for legged manipulation.

**[Learning Tactile-Aware Quadrupedal Loco-Manipulation Policies](https://arxiv.org/abs/2604.27224)**
Already covered under "Dexterous Manipulation". Touch-aware loco-manipulation.

**[PAKE: Learning Whole-Body Loco-Manipulation with Partial Kinematic Embeddings](https://arxiv.org/abs/2607.11041)**
Introduces a whole-body control framework that decomposes loco-manipulation into partial reference motion generation and low-level imitation control. Uses a Kinematic Normalizing Flow model to generate feasible motions, achieving low tracking errors on a quadruped with a 6-DoF arm.

### Foundation Models & Agentic Frameworks

**[SWIFT: A Small-World Interaction Framework for Flow-Aware Trajectory Prediction in Autonomous Driving](https://arxiv.org/abs/2607.09741)**
A trajectory prediction framework integrating small-world networks with traffic flow theory. Outperforms baselines on nuScenes, MoCAD, and NGSIM, showing improved generalization under distribution shifts.

**[LLM-Centric Agentic AI for UAV Swarms: Architecture, Enabling Technologies, and Open Problems](https://arxiv.org/abs/2607.09756)**
A survey and proposed architecture for LLM-centric agentic AI in UAV swarms. Covers key technologies, including onboard computing, 5G/6G connectivity, and analyzes threats like Priority Manipulation Attacks.

**[Embodied-R1.5: Evolving Physical Intelligence via Embodied Foundation Models](https://arxiv.org/abs/2606.11324)**
Introduces a unified embodied foundation model integrating reasoning capabilities. Using automated data pipelines and multi-task RL, it achieves SOTA on 16/24 embodied VLM benchmarks and can be fine-tuned into a VLA outperforming leading models.

**[A Glimpse into Long-term Physical Coexistence with Intelligent Robots](https://arxiv.org/abs/2607.11377)**
Introduces PHILIA, a multi-robot agent built around a robot gateway abstraction for long-term physical coexistence. Decouples high-level semantic reasoning from low-level execution for plug-and-play integration of various components.

**[From World Action Models to Embodied Brains: A Roadmap for Open-World Physical Intelligence](https://arxiv.org/abs/2607.11689)**
Reviews the evolution toward World Action Models (WAMs) and proposes a roadmap for physical intelligence centered on an "embodied brain" that integrates multimodal context for physical reasoning.

**[NeuralActuator: Neural Actuation Modeling for Robot Dynamics and External Force Perception](https://arxiv.org/abs/2607.11734)**
A neural actuator model that jointly predicts generalized-effort, external forces, and motor condition. Introduces the Neural Actuation Dataset (NAD) collected with a twin-arm teleoperation system.

**[ABot-N1: Toward a General Visual Language Navigation Foundation Model](https://arxiv.org/abs/2607.10383)**
A foundation model for Visual Language Navigation that decouples cognition from control via a slow-fast architecture. Uses a fast action expert to generate continuous waypoints guided by a slow vision-language reasoner's pixel goals.

**[Autonomous Close-Proximity Photovoltaic Panel Coating Using a Quadcopter](https://arxiv.org/abs/2509.10979)**
A quadcopter system for autonomous coating of PV panels, using visual-inertial odometry and model-based control that accounts for ground effect and mass decrease during liquid dispersion.

**[Calf-Integrated Arms for Bimanual Quadruped Loco-Manipulation](https://arxiv.org/abs/2607.06186)**
An integrated design where a manipulator is built into each front calf of a Unitree Go2, enabling bimanual manipulation while all four feet stay planted. A VLM sequences skills for long-horizon autonomy.

**[WristMimic: Full-Body Humanoid Control with Wrist-Guided Manipulation](https://arxiv.org/abs/2607.06438)**
A wrist-guided whole-body control framework for humanoid manipulation. Explicitly separates contact-free body motion from contact-rich hand manipulation by using the wrist as a natural gate between the two regimes.

**[Task-Conditioned Synthetic Data Generation for Improving ML Performance...](https://arxiv.org/abs/2607.09751)**
Proposes a Task-Conditioned synthetic data generation algorithm for agriculture, augmenting real data to improve downstream ML performance. Demonstrates significant improvements across multiple prediction tasks.
