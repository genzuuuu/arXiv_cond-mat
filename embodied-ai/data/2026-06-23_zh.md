### 1. 具身智能基础模型与规划

*   **[Vesta: A Generalist Embodied Reasoning Model](https://arxiv.org/abs/2606.20905)**: 提出了一个名为 Vesta 的统一具身基础模型，将定位、空间推理、导航和长时程规划整合到一个单一模型中。通过大规模策划的语料库和简单的多模态记忆模块，Vesta 在多个基准测试上平均优于单个最强基线模型超过 20%。

*   **[MemoryVAM: Integrating Memory into Video Action Model for Robot Manipulation](https://arxiv.org/abs/2606.20679)**: 针对长时程操作任务中的非马尔可夫特性，提出了 MemoryVAM。它引入了一个情节记忆机制（Recap-Cue 模块），利用感知器网络压缩历史信息为记忆令牌，并注入到视频骨干网络和动作解码器中，极大地提升了长时程任务的性能（在 LIBERO-Mem 上平均成功率从 5% 提升至 42.5%）。

*   **[UniviewVLA: A Unified Multiview Vision-Language-Action Model with World Modeling](https://arxiv.org/abs/2606.21501)**: 提出了 UniviewVLA，一个统一的多视角 VLA 模型。通过一个世界模型，它可以从标准的两摄像头输入推断出多视角的未来场景演化，从而揭示遮挡信息。在遮挡任务上，成功率从 40% 提升至 73.3%。

*   **[KITE: Decoupling Kinematics and Interaction for Zero-Shot Cross-Embodiment Manipulation](https://arxiv.org/abs/2606.22113)**: 提出了 KITE，一个解耦运动学和交互的零样本跨本体操作框架。它将操作分解为与本体无关的任务推理（通过接触模式学习潜在交互意图）和与本体相关的运动控制。新本体只需学习一个意图条件动作解码器，无需重新收集演示数据。

### 2. 灵巧手操作与抓取

*   **[A Human-Inspired Thumb-Index Robotic Hand with Strain Gauges Embedded in Soft Joints](https://arxiv.org/abs/2606.21245)**: 介绍了一种受人类启发、仅包含拇指和食指的轻量级低成本机器人手（STIR Hand）。该手部采用欠驱动、腱驱动设计，并在硅胶软关节中嵌入了柔性应变片，实现了被动抓取适应和自身/外部感知。

*   **[DexTeleop-0: Force-Aware Bimanual Dexterous Teleoperation with Ego-Centric Perception towards Shared Autonomy](https://arxiv.org/abs/2606.23431)**: 提出了一种面向力觉意识的双臂灵巧遥操作系统 DexTeleop-0。通过一个实时的触觉驱动优化循环，将粗略的人类意图转化为精确、力顺应性的机器人指令，填补了高精度接触丰富任务的遥操作鸿沟。

*   **[AutoDex: An Automated Real-World System for Dexterous Grasping Data Collection](https://arxiv.org/abs/2606.23689)**: 提出了 AutoDex，一个自动化的真实世界灵巧抓取数据收集系统。它可以自动执行物体定位、抓取执行、成败标注和场景重置等流程，无需人工干预。相比于遥操作，其数据收集效率提升了 4.8 倍。

*   **[T-Rex: Tactile-Reactive Dexterous Manipulation](https://arxiv.org/abs/2606.17055)**: 提出了 T-Rex，一个触觉反应型的灵巧操作框架。通过构建一个100小时的大规模触觉数据集的，并提出一个可变速率混合变换器架构来处理高频触觉信号，在需要精细力控的12个操作任务上，平均成功率比最强基线高出30%以上。

*   **[EgoInfinity: A Web-Scale 4D Hand-Object Interaction Data Engine for Any-View Robot Retargeting and Video-to-Action Robot Learning](https://arxiv.org/abs/2606.17385)**: 提出了 EgoInfinity，一个可从互联网视频中生成 4D 手物交互数据的引擎。它能够将视频中的手部动作重定向到不同形态的机器人上，实现视频到动作的机器人学习，解决了真实机器人训练数据规模受限的问题。

### 3. 操作任务与策略学习

*   **[R2HandoverSim: A Simulation Framework and Benchmark for Robot-to-Human Object Handovers](https://arxiv.org/abs/2606.21011)**: 提出了 R2HandoverSim，一个用于评估机器人向人类递送物体的仿真基准。通过 30 人的用户研究，证明了仿真结果与真实世界评估具有相关性，并强调了规划可行性、可达性等多项指标的重要性。

*   **[Geometric Entropy: When Trajectory Diversity Helps and Hurts in Imitation Learning](https://arxiv.org/abs/2606.20871)**: 研究了演示数据中轨迹形状多样性对模仿学习的影响。引入了几何熵作为衡量指标，发现成功率与几何熵呈倒U型关系：适度增加多样性有益，但过高会导致策略模糊，降低性能。

*   **[Wh0: Generative World Models as Scalable Sources of Egocentric Human Hand Manipulation Data](https://arxiv.org/abs/2606.22136)**: 提出 Wh0 框架，利用生成式视频世界模型作为可扩展且可控的自我中心人类手部操作数据来源。通过生成 5 万条手物交互视频数据，并将其转化为机器人可训练的监督信号，在18个灵巧操作任务上，零样本成功率从 8.3% 提升至 38.9%。

*   **[Flow as Flow: Modeling Robot Velocity Fields as Probability Velocity Fields for Flow-Based Object Manipulation](https://arxiv.org/abs/2606.23090)**: 提出 Flow as Flow 框架，将机器人流（机器人速度场）建模为基于流匹配的概率流。这种方法能高效地生成高质量的动作，并在13个现实操作任务上实现了比基线方法更高的平均成功率。

*   **[LaST-HD: Learning Latent Physical Reasoning from Scalable Human Data for Robot Manipulation](https://arxiv.org/abs/2606.23685)**: 提出 LaST-HD，一种从人类手部演示中学习潜在物理推理的框架。通过对齐人类和机器人在共享潜在推理空间中的表征，使得机器人能够从人类手部数据中学习高效的物理交互，并在新物体和场景中具备更强的泛化能力。

### 4. 触觉感知与融合

*   **[Tactile Genesis: Exploring Tactile Sensors at Scale for Learning Dexterous Tasks](https://arxiv.org/abs/2606.22332)**: 提出了 Tactile Genesis，一个 GPU 并行的触觉传感器仿真平台。它支持多种触觉信号（如接触力、力矩、位移场等）。研究发现，传感器放置位置远重要于类型，覆盖全手的触觉比仅在指尖的触觉更有效，且力/力矩信号最有用。

*   **[HapTile: A Haptic-Informed Vision-Tactile-Language-Action Dataset for Contact-Rich Imitation Learning](https://arxiv.org/abs/2606.04825)**: 发布了 HapTile 数据集，这是首个在遥操作中为操作员提供触觉反馈的视触觉-语言-动作数据集，用于接触丰富的操作任务学习。该数据集包含指尖触觉和触觉反馈驱动的演示，为学习精细的力控策略提供了宝贵资源。

*   **[FTP-1: A Generalist Foundation Tactile Policy Across Tactile Sensors for Contact-Rich Manipulation](https://arxiv.org/abs/2606.13102)**: 提出了 FTP-1，首个通用基础触觉策略，能够跨不同触觉传感器和本体进行预训练和迁移。它通过在26个数据源上预训练，学会了触觉操作技能，并能在未见过的新型传感器上提升 31% 的成功率。

*   **[Geometric Reconstruction of Extrinsic Contact Trajectories using Tactile Sensing and Proprioception for Tool Manipulation](https://arxiv.org/abs/2606.22251)**: 研究了如何利用抓取层面的触觉感知和机器人本体感觉，重建工具与外部环境接触点的轨迹。在单点接触假设下，实现在线重建，轨迹均方根误差为 8.59 毫米。

### 5. 遥操作与 Sim-to-Real

*   **[Duet: Dual-Robot Understanding via Efficient Teaching](https://arxiv.org/abs/2606.20990)**: 提出了 DUET，一个用于双机器人自主协作的学习框架。通过统一的 VR 遥操作平台和人类协作先验的引导，使用动作分块变换器学习协同策略。利用人类先验数据，将遥操作数据收集效率平均提升了 5.4 倍。

*   **[Real-World Deployment of Massively Parallel Sampling-Based MPC for Contact-Rich Manipulation](https://arxiv.org/abs/2606.20712)**: 本文介绍了将基于采样的模型预测控制（SMPC）部署到真实机器人上进行接触丰富操作的全流程。通过在 JAX 和 MuJoCo MJX 中进行大规模并行仿真训练，并在 Franka Research 3 机器人上成功执行了 Push-T 任务，验证了仿真到真实迁移的可行性。

*   **[Sim-to-Real Visuomotor Learning for Surgical Robotics](https://arxiv.org/abs/2606.21396)**: 展示了一个在仿真中训练的视觉运动策略，如何在 sim-to-real 迁移后，成功部署在达芬奇研究套件上，并克服了其固有的运动学不准确性问题。策略通过视觉反馈主动补偿硬件误差。

*   **[ALOE: Action-Level Off-Policy Evaluation for Vision-Language-Action Model Post-Training](https://arxiv.org/abs/2602.12691)**: 提出 ALOE，一个用于 VLA 模型后训练的动作级离策略评估框架。它通过组合分块时序差分和保守值聚合，实现对当前策略的稳定评估，从而指导更有效的策略改进。在真实世界的长时程和高精度任务（如手机打包、衣物折叠）上验证了其有效性。

### 6. 其他相关

*   **[Any-Mug: Pose-Agnostic Robotic Functional Grasping](https://arxiv.org/abs/2606.21148)**: 提出 AnyMug，一个用于功能性抓取的标准化的视觉运动强化学习框架。它通过将观测和动作标准化到物体中心坐标系，实现了对任意位姿下的杯子的功能抓取，并实现了零样本的 sim-to-real 迁移。

*   **[CoorDex: Coordinating Body and Hand Priors for Continuous Dexterous Humanoid Loco-Manipulation](https://arxiv.org/abs/2606.23680)**: 提出了 CoorDex，一个能够在人形机器人上实现边运动边进行灵巧操作的流水线。它将身体和手部的高维控制转化为协调的潜在残差控制，解决了全身运动和手指级接触的联合优化问题。

*   **[DeformX: A Versatile Co-Simulation Framework for Deformable Linear Objects](https://arxiv.org/abs/2606.22116)**: 介绍了 DeformX，一个专门用于仿真可变形线性物体的协同仿真框架。它将 Cosserat 杆物理引擎与 Isaac Sim 集成，兼顾了物理真实性和视觉逼真度，并展示了其强大的 sim-to-real 迁移能力。

*   **[FOCA: Future-Oriented Conditioning for Data-Efficient VLA Adaptation](https://arxiv.org/abs/2606.20867)**: 提出了 FOCA，一个面向未来预测的少样本 VLA 适应框架。它通过预测任务相关的未来交互嵌入并与未来目标观测对齐，实现了在极少量演示下的高效学习。在 LIBERO 上仅用 20 个演示即达到 95.7% 的成功率。

*   **[CoRDE: Concept-Prior Routed Diffusion Experts for Structural Generalization in Robot Manipulation](https://arxiv.org/abs/2606.21935)**: 提出了 CoRDE，一个结构引导的变分蒸馏框架。它利用冻结的概念编码器提取语义分布，并通过学习到的软映射矩阵引导扩散专家路由，有效解决了多任务和长时程场景下的梯度冲突和路由崩溃问题。

*   **[Temporal Logic Guidance for Action-Only Diffusion Policies with World Models](https://arxiv.org/abs/2606.22729)**: 提出了一种引导动作-扩散策略的方法。通过一个单独的世界模型和信号时序逻辑（STL）公式，在推理时引导扩散过程，使得策略在满足约束的同时保持任务性能。

*   **[FlowDPG: Deterministic Policy Gradient on Flow Matching Policies for Real-World Manipulation](https://arxiv.org/abs/2606.22303)**: 提出 FlowDPG，一个专门为流匹配策略设计的 DDPG 风格强化学习方法。它通过蒸馏技巧避免了在流匹配策略中需要昂贵的通过时间反向传播的问题，并在真实世界的双臂 AirPods 装配任务中取得了 92% 的成功率。

*   **[Scalable Multi-Task Data Generation via Reinforcement Learning for Language-Conditioned Bimanual Dexterous Manipulation](https://arxiv.org/abs/2606.22471)**: 提出了一种基于强化学习的可扩展数据生成方法。通过在仿真中利用通用奖励设计和域随机化，自动合成高质量的双臂灵巧操作数据，并用于训练语言条件化多任务策略，显著提升了模型的泛化能力。
