### Video & World Models for Policy Learning

**[BIFROST: Bridging Invariant Feature Representation for Observation-space Sim2Real Transfer](https://arxiv.org/abs/2607.01410) BIFROST: Bridging Invariant Feature Representation for Observation-space Sim2Real Transfer**
[Link](https://arxiv.org/abs/2607.01410)
Proposes learning a shared latent representation from paired cross-domain data (sim & real) using a cross-domain bisimulation objective. A policy trained on these latent states in simulation achieves zero-shot sim-to-real transfer on contact-rich manipulation and visual servoing tasks, outperforming domain adaptation and co-training baselines.

**[ACID: Action Consistency via Inverse Dynamics for Planning with World Models](https://arxiv.org/abs/2607.02403) ACID: Action Consistency via Inverse Dynamics for Planning with World Models**
[Link](https://arxiv.org/abs/2607.02403)
Enhances decision-time planning with world models by introducing cycle action consistency. An inverse dynamics model checks if the action inferred backward from a predicted transition matches the original action, using this residual as a cost. Improves planning across manipulation, articulated control, and navigation tasks.

**[WorldSample: Closed-loop Real-robot RL with World Modelling](https://arxiv.org/abs/2607.02431) WorldSample: Closed-loop Real-robot RL with World Modelling**
[Link](https://arxiv.org/abs/2607.02431)
A physically grounded data augmentation framework for real-world RL. It closes a loop between real rollouts, world-model generation, and policy improvement. Post-trained world models generate high-fidelity synthetic transitions, and Policy-Paced Learning regulates their use, improving success rate by 28% and reducing training steps by 59%.

**[VT-WAM: Visual-Tactile World Action Model for Contact-Rich Manipulation](https://arxiv.org/abs/2607.02503) VT-WAM: Visual-Tactile World Action Model for Contact-Rich Manipulation**
[Link](https://arxiv.org/abs/2607.02503)
A world-action model that jointly learns visual prediction, tactile deformation prediction, and action prediction in a unified flow matching framework for contact-rich manipulation. Uses asymmetric Mixture-of-Transformers attention and contact-gated attention to improve performance on tasks requiring tactile feedback, achieving a 71.67% average success rate across six real-world tasks.

**[Transport Discrepancy as a Reliability Signal for Vision-Language-Action Models](https://arxiv.org/abs/2512.01715) Transport Discrepancy as a Reliability Signal for Vision-Language-Action Models**
[Link](https://arxiv.org/abs/2512.01715)
Introduces DiG, a plug-in module for flow-matching VLA policies that uses the optimal transport cost between backbone features and the action expert's input as a reliability signal. This per-step estimate is used to modulate a residual feature refinement and training loss, improving success rates under distribution shift and on long-horizon tasks.

**[DVG-WM: Disentangled Video Generation Enables Efficient Embodied World Model for Robotic Manipulation](https://arxiv.org/abs/2606.32028) DVG-WM: Disentangled Video Generation Enables Efficient Embodied World Model for Robotic Manipulation**
[Link](https://arxiv.org/abs/2606.32028)
Proposes a world model that explicitly decomposes the task into dynamics learning and visual synthesis. It first generates intermediate visual states to preview physical interaction, then refines them into high-fidelity videos. Achieves up to 3.97x acceleration in video generation for manipulation tasks.

**[PhysMani: Physics-principled 3D World Model for Dynamic Object Manipulation](https://arxiv.org/abs/2607.01938) PhysMani: Physics-principled 3D World Model for Dynamic Object Manipulation**
[Link](https://arxiv.org/abs/2607.01938)
Couples a physics-principled 3D Gaussian world model with a future-aware action policy for manipulating fast-moving targets. The world model learns a divergence-free velocity field for physically grounded future dynamics, and the policy integrates these predictions via cross-attention. Outperforms baselines on a 16-task dynamic manipulation benchmark.

**[Bridge-WA: Predicting Where and How the World Changes for Robotic Action](https://arxiv.org/abs/2607.02195) Bridge-WA: Predicting Where and How the World Changes for Robotic Action**
[Link](https://arxiv.org/abs/2607.02195)

Improves task success and robustness, especially under out-of-distribution visual shifts, without the cost of dense future rollout generation.

**[SUNTA: Hierarchical Video Prediction with Surprise-based Chunking](https://arxiv.org/abs/2607.02087) SUNTA: Hierarchical Video Prediction with Surprise-based Chunking**

**[WorldOdysseyBench: An Open-World Benchmark for Long-Horizon Stability of Interactive World Models](https://arxiv.org/abs/2606.31672) WorldOdysseyBench: An Open-World Benchmark for Long-Horizon Stability of Interactive World Models**
