### Tactile Sensing & Dexterous Manipulation

- **[VibeCheck: Using Active Acoustic Tactile Sensing for Contact-Rich Manipulation](https://arxiv.org/abs/2504.15535)**
  Presents an active acoustic sensing gripper that uses piezoelectric fingers to send and receive vibrations through objects, enabling classification, grasping position estimation, and contact type detection. Used with a learned policy for peg insertion tasks.
- **[TaCauchy: An Extensible FEM Framework for Vision-Based Tactile Simulation](https://arxiv.org/abs/2606.20426)**
  A high-fidelity FEM simulation framework for vision-based tactile sensors in Isaac Sim. Computes Cauchy stress tensors for contact forces and pressure distributions, achieving strong agreement with real sensor data.
- **[PTLD: Sim-to-real Privileged Tactile Latent Distillation for Dexterous Manipulation](https://arxiv.org/abs/2603.04531)**
  Addresses the sim-to-real gap for tactile manipulation by using privileged sensors in the real world to collect data and distill a robust tactile state estimator. Achieves a 182% improvement over proprioception-only policies on in-hand rotation tasks.
- **[Belt-Finger: An Affordable Soft Belt-Driven Gripper for Dexterous In-Hand Manipulation](https://arxiv.org/abs/2606.20193)**
  A low-cost upgrade for parallel-jaw grippers that uses soft belts to add three in-hand degrees of freedom (translation, pitch, roll). Demonstrated with MPC, teleoperation, and trained policies for challenging manipulation tasks.

### Visuomotor & Imitation Learning

- **[One Demo is Worth a Thousand Trajectories: Action-View Augmentation for Visuomotor Policies](https://arxiv.org/abs/2606.19586)**
  An augmentation framework using Gaussian Splatting and trajectory optimization to generate visually realistic, physically feasible demonstrations from a single real-world capture. Improves policy success rates in scenes with unseen obstacles.
- **[Pose6DAug: Physically Plausible Multi-view Object Swapping for Robot Data Augmentation](https://arxiv.org/abs/2606.20118)**
  A failure-driven data augmentation framework that creates new, physically grounded demonstrations by splicing the trajectory from a successful episode with a 3D mesh of a novel object.  Improves VLA policy generalization to new objects.
- **[MirrorDuo: Reflection-Consistent Visuomotor Learning from Mirrored Demonstration Pairs](https://arxiv.org/abs/2606.20048)**
  A data augmentation strategy that generates mirrored counterparts for each demonstration. This "collect one, get one for free" approach improves performance under the same data budget and enables efficient skill transfer across workspace sides.
- **[EquiVLA: A General Framework for Rotationally Equivariant Vision-Language-Action Models](https://arxiv.org/abs/2606.19784)**
  Introduces geometric inductive biases to VLA models by making them approximately SO(2)-equivariant. Achieves significant performance gains on LIBERO, CALVIN, and real-world Mobile ALOHA tasks.
- **[Finetuning Vision-Language-Action Models Requires Fewer Layers Than You Think](https://arxiv.org/abs/2606.20246)**
  A training-free compression method for VLA models that identifies and removes redundant layers, reducing model depth by up to 50%. This leads to a 40-50% reduction in fine-tuning time and up to 30% faster inference.
- **[Temporal Self-Imitation Learning](https://arxiv.org/abs/2606.19752)**
  An RL framework that mines temporally efficient successful trajectories from the agent's own experience and uses them as supervision for future learning, improving sample efficiency across 15 long-horizon manipulation tasks.
- **[Start Right, Arrive Right: Asynchronous Execution via Initial Noise Selection](https://arxiv.org/abs/2606.19774)**
  Addresses latency issues in action-chunking policies by reframing asynchronous inference as a noise selection problem. The method (PAINT) improves execution consistency in real-world single-arm, bimanual, and humanoid tasks.
- **[Frequency-Aware Flow Matching for Continuous and Consistent Robotic Action Generation](https://arxiv.org/abs/2606.20135)**
  A flow-matching policy that outputs continuous, temporally consistent actions by operating in the frequency domain using DCT and regularizing the temporal derivative to ensure smoothness.
- **[DiffusionVS: A Generative Framework for Robust Visual Servoing Based on Diffusion Policy](https://arxiv.org/abs/2606.19397)**
  Applies Diffusion Policy to visual servoing, predicting action sequences for smooth and robust camera velocity control. Achieves near-perfect success in simulation and 93% in real-world experiments.
- **[Playful Agentic Robot Learning](https://arxiv.org/abs/2606.19419)**
  Introduces a framework where an embodied coding agent engages in self-directed play to learn a library of code-based skills. These skills are then reused to solve novel downstream tasks, showing significant improvements over baselines.

### Robot Hand & Embodiment Design

- **[Generating Robot Hands from Human Demonstrations](https://arxiv.org/abs/2606.20549)**
  A data-driven framework that optimizes robot hand designs by using large-scale human fingertip motion data to reproduce target motions. The framework produced both a 6-DoF general-purpose hand and lower-DoF task-specific hands with print-in-place fabrication.

### Teleoperation & Human-Robot Interaction

- **[CoLI: A Reproducible Platform for Continuum Robot Learning via Monolithic 3D Printing and Isomorphic Teleoperation](https://arxiv.org/abs/2606.20389)**
  An open-source, 3D-printed continuum robot with an isomorphic teleoperation interface. The platform enables imitation-learning-based autonomous control, addressing reproducibility and control challenges in continuum robotics.
- **[TASC: Task-Aware Shared Control for Relational Telemanipulation](https://arxiv.org/abs/2509.10416)**
  A shared control framework for telemanipulation that infers user intent from motion-only input and provides assistance for both grasping and object interaction, guided by a vision-language model.

### World Models & Planning for Manipulation

- **[ImageWAM: Do World Action Models Really Need Video Generation, or Just Image Editing?](https://arxiv.org/abs/2606.19531)**
  Proposes ImageWAM, a WAM that repurposes pretrained image editing models for robot action prediction, sidestepping the computational cost and errors of video generation. It achieves comparable or better performance with 1/6 the FLOPs and 1/4 the latency of video-based WAMs.
- **[MemoryWAM: Efficient World Action Modeling with Persistent Memory](https://arxiv.org/abs/2606.20562)**
  A world action model with a hybrid memory design (recent frames, event anchors, gist tokens) for efficient long-horizon, memory-dependent manipulation tasks. Outperforms VLA and WAM baselines while maintaining computational efficiency.
- **[FlowMaps: Modeling Long-Term Multimodal Object Dynamics with Flow Matching](https://arxiv.org/abs/2606.20209)**
  A latent flow matching model that predicts multimodal distributions over future object locations in changing environments. Used for dynamic Object Navigation, it significantly improves robot search performance.

### Benchmarking & Data Standards

- **[WorkBenchMark: A LEGO-Based Assembly Benchmark with an Assembly-by-Disassembly Baseline](https://arxiv.org/abs/2606.19358)**
  A LEGO Duplo-based benchmark for robotic assembly, providing 400 tasks across four tiers of complexity. An open-vocabulary, planning-based baseline significantly outperforms a modern VLA approach.
- **[Humanoid Everyday: A Comprehensive Robotic Dataset for Open-World Humanoid Manipulation](https://arxiv.org/abs/2510.08807)**
  A large-scale humanoid manipulation dataset with 10.3k trajectories across 260 tasks featuring multimodal data. Introduces a cloud-based evaluation platform for policy benchmarking.
- **[Data Standards for Humanoid Robotics: The Missing Infrastructure for Physical AI](https://arxiv.org/abs/2606.19769)**
  Argues that data standards are foundational for Physical AI, enabling embodied experience to be interpretable, shareable, and reusable across robots. Discusses ISO/WD 26264-1.

### Sim-to-Real & Simulation

- **[Physical Atari: A Robust and Accessible Platform for Real-time Reinforcement Learning on Robots](https://arxiv.org/abs/2606.19357)**
  A low-cost, robust platform using a robotic arm to play Atari, bridging the gap between simulation and the physical world. Demonstrates the need for on-device adaptation under distribution shifts.
- **[Simulating Robotic Locomotion in Sand: Resistive Force Theory in an Open-Source Physics Engine](https://arxiv.org/abs/2606.19504)**
  Integrates 3D Granular Resistive Force Theory into the MuJoCo physics engine for simulating robot locomotion on sand. Predicts walking performance and foot sinkage within 20% of real-world experiments.
- **[DF-ExpEnse: Diffusion Filtered Exploration for Sample Efficient Finetuning](https://arxiv.org/abs/2606.19656)**
  An exploration technique that uses the multimodality of a pretrained generative control policy to create a candidate set of actions, then uses an ensemble of critics to select the most informative one, improving fine-tuning sample efficiency.

### Multi-Agent & Task Planning

- **[ENPIRE: Agentic Robot Policy Self-Improvement in the Real World](https://arxiv.org/abs/2606.19980)**
  A framework where coding agents autonomously improve dexterous manipulation policies in the real world by resetting, executing, verifying, and refining. Achieves 99% success on challenging tasks.
- **[One-to-Two Acting: A Novel Framework for Single-arm Agent Action Expansion to Dual Arms](https://arxiv.org/abs/2606.19897)**
  A hierarchical framework ExS2D that enables dual-arm manipulation by expanding a single-arm policy, using a VLM coordinator for precedence-aware action allocation, requiring zero bimanual demonstrations.
- **[Co-VLA: Coordination-Aware Structured Action Modeling for Dual-Arm Vision-Language-Action Systems](https://arxiv.org/abs/2606.20285)**
  Introduces explicit structural priors into VLA models for bimanual coordination. A modular loss shapes shared and residual latents for task-level coordination and arm-specific execution, improving performance in tight-coordination tasks by 27%.
- **[Fail-RAG : A Retrieval Augmented Generation Informed Framework for Robot Failure Identification](https://arxiv.org/abs/2606.19598)**
  A RAG-based failure detection system that uses VLMs to analyze failures by querying a database of failure images and context. Achieves 25% higher detection accuracy than off-the-shelf VLMs on warehouse automation tasks.
- **[Co-policy: Responsive Human-Robot Co-Creation for Musical Performances](https://arxiv.org/abs/2606.19914)**
  A framework for human-robot musical co-creation that separates semantic intent grounding from visuomotor execution. Uses a fine-tuned VLM planner and a GMP for real-time, physically grounded action generation.
