### 1. 抓取与灵巧操作
*   **[Predicting Grasping Compliance in Robotic Hands through Analytical-Model-Informed Neural Networks](https://arxiv.org/abs/2607.17541)**：提出一种分析模型驱动的神经网络（AMINN），用于预测欠驱动机械手在受到外力时，被抓取工具的形变和稳定性。该方法结合了分析力学和数据驱动学习，在预测抓取顺应性方面表现出色，且物理一致性更好。
*   **[GraspADMM: Improving Dexterous Grasp Synthesis via ADMM Optimization](https://arxiv.org/abs/2603.13832)**：提出GraspADMM框架，利用交替方向乘子法（ADMM）解耦目标接触点和手部姿态的优化，从而在灵巧抓取合成中兼顾多样性和物理可行性，显著提升了抓取成功率，尤其在极端低摩擦条件下。
*   **[Stochastic Entanglement of Deterministic Origami Tentacles For Robust Robotic Grasping](https://arxiv.org/abs/2604.26897)**：介绍了一种基于折纸原理的触手夹爪，通过精心设计的折痕和拉线，使触手在拉动时能确定性盘绕，而多个触手的随机纠缠则能稳健抓取任意形状的物体，设计简单且控制成本低。
*   **[A Sensorless, Inherently Compliant Anthropomorphic Musculoskeletal Hand Driven by Electrohydraulic Actuators](https://arxiv.org/abs/2603.24357)**：设计了一款由远程Peano-HASEL软体驱动器驱动的类人肌肉骨骼手。通过监测电流实现无传感器抓取检测，且其固有的力限制特性提供了物理柔顺性，能无损抓取易碎物体。
*   **[Configuration-Induced Passive Self-Rotation for Perception-Enhanced Autonomous Flight](https://arxiv.org/abs/2607.17646)**：提出一种利用三旋翼飞行器结构设计引起的被动自旋转，以扩大机载传感器视野。该方法无需额外传感器即可在狭窄杂乱环境中实现感知增强的自主飞行。
*   **[Seg2Grasp: A Robust Modular Suction Grasping in Bin Picking](https://arxiv.org/abs/2607.17757)**：提出了一个模块化的吸盘抓取流水线Seg2Grasp，通过分割、抓取和分类三个步骤，实现对动态和杂乱料箱中未知物体的稳健吸盘抓取。

### 2. 触觉感知与肌电控制
*   **[FM-VLA: Force-based Memory for Vision-Language-Action Models in Contact-Rich Manipulation](https://arxiv.org/abs/2607.18231)**：提出FM-VLA模型，引入基于力的记忆（力记忆token），使视觉-语言-动作模型能利用历史接触事件进行时序上下文推理，解决了纯视觉记忆在接触密集任务（如按按钮、擦盘子）中失效的问题。
*   **[Feeling the Unexpected: ResTacVLA for Contact-Rich Manipulation via Residual Tactile Representation](https://arxiv.org/abs/2607.03387)**：受预测编码启发，提出ResTacVLA。该方法将触觉信息重新定义为“残余触觉表示”，即视觉先验与实际物理感觉之间的差异，从而在视觉-语言-动作模型中有效整合触觉，防止多模态坍缩。
*   **[Real-Time sEMG-Based Telecontrol of an Assistive Robotic Arm Using a 1D Convolutional Neural Network](https://arxiv.org/abs/2607.16310)**：实现了一个基于四通道表面肌电信号和一维卷积神经网络的实时遥操作系统，用于控制辅助机械臂，测试准确率超过90%，延迟约0.32秒。
*   **[Thermopneumatic Pixels: Fast, Localized, Robust, Low-Voltage Touch Feedback](https://arxiv.org/abs/2603.16750)**：介绍了一种热气压像素（TPP）技术，能够提供快速、局部化、低电压驱动的动态触觉反馈，为机器人触觉传感和交互界面提供了新思路。

### 3. 视觉-运动策略与模仿学习
*   **[Patch Policy: Efficient Embodied Control via Dense Visual Representations](https://arxiv.org/abs/2607.18236)**：提出Patch Policy，一种轻量级的策略架构，通过块状因果注意力掩码直接消费预训练的密集块（Patch）特征。该方法性能显著优于使用全局池化特征的方法，参数仅为大型VLA模型的0.7%，却实现了更高的成功率。
*   **[Foresight Residual RL for Long-Horizon Robot Manipulation with Vision-Language-Action Models](https://arxiv.org/abs/2607.16506)**：针对长时域操作任务中“下游步骤成功率低”的问题，提出了“预见性残差强化学习”。该算法通过离线评估的“预见值”来奖励当前子任务的终止状态，优化任务间的交接质量，在三阶段装配任务中显著提升了成功率。
*   **[TempoVLA: Learning Speed-Controllable Vision-Language-Action Policies](https://arxiv.org/abs/2606.06491)**：提出TempoVLA，一个动作速度可控的VLA模型。通过数据增强和模型端条件机制，实现对机器人执行速度的灵活控制，可在低风险阶段加速、高风险阶段减速。
*   **[Asynchronous Multimodal Diffusion Policy Composition via Latency-Aware Guidance Fusion](https://arxiv.org/abs/2607.17257)**：针对多模态融合中不同传感器采样率和推理延迟不同的问题，提出LAG-Fusion框架。该框架允许各个模态的策略以自身原生速率运行，并通过延迟感知的引导融合进行异步组合，提升了接触操作中的策略响应速度和任务性能。
*   **[Learning Stable In-Grasp Manipulation in a Non-Dropping Action Space (https://arxiv.org/abs/2606.28196)**：将灵巧操作技能分解为更简单的、可分析组件，并利用经典物理和控制理论的约束和指导进行强化学习，从而实现稳定且高效的物体手中操作技能学习。
*   **[HCPG-Flow:Hierarchical Contact-Progress Guidance for Flow-Policy Robot Manipulation](https://arxiv.org/abs/2607.17651)**：提出HCPG-Flow，一个分析性的在线动作选择器。它通过分层的、以物体为中心的接触-进程导向信号，在扩散策略的候选动作中选择最佳执行动作，有效提升了多模态动作分布下的操作成功率。

### 4. 遥操作、模仿学习与Sim-to-Real
*   **[MEVION: Low-Cost Open-Source Data Collection System for Powerful and High-Speed Dual-Arm Manipulation](https://arxiv.org/abs/2607.17970)**：介绍MEVION，一个低成本、开源的双臂机器人数据采集系统。它通过特有的结构设计，能提供更大的力和速度，使其可以处理重物和进行快速操作，从而扩展了模仿学习的数据采集范围。
*   **[ICLR: In-Context Imitation Learning with Visual Reasoning](https://arxiv.org/abs/2603.07530)**：提出ICLR框架，通过在演示提示中增加结构化的视觉推理轨迹（如预期未来轨迹），使机器人能从少量演示中更好地泛化到新任务和新物体配置。
*   **[Isaac Sim-to-Real: Reinforcement Learning based Locomotion for Quadrupeds](https://arxiv.org/abs/2607.18135)**：展示了利用NVIDIA Isaac Sim和Isaac Lab进行四足机器人运动控制的强化学习训练，并实现了零样本sim-to-real迁移，在Unitree Go1机器人上验证了其性能。
*   **[World Translation: Minimizing Sim-to-Real Gap with Backward Dynamics Extraction and Unpaired Domain Translation](https://arxiv.org/abs/2607.18154)**：提出“世界翻译”方法，通过从观察到的状态-动作转移中逆向提取动力学信息，并将其作为特征进行无配对域翻译，以弥补sim-to-real差距，在多种机器人平台上取得了更好的效果。
*   **[CopGrasp: Cooperative Grasping with Human and Dexterous Robotic Hands](https://arxiv.org/abs/2607.17541)** (注意：此论文实际是[Predicting Grasping Compliance in Robotic Hands through Analytical-Model-Informed Neural Networks](https://arxiv.org/abs/2607.17541)，但摘要内容更贴近*抓取与灵巧操作*类别，故分类到[HyperDCM: Dynamic Cluster Memory Replay in Hyperbolic Space for Continual Robotic Navigation Across Scenes](https://arxiv.org/abs/2607.16267). 已修正)

### 5. 操作规划与推理
*   **[ConceptTree: Bringing Semantic Transparency to Black-Box Decision Making for Robotic Manipulation](https://arxiv.org/abs/2607.17861)**：提出ConceptTree框架，将机器人高层技能选择转化为对人类可解释概念（如物体状态、空间关系）的推理，使决策过程透明且可干预。
*   **[PrimitiveVLA: Learning Reusable Motion Primitives for Efficient and Generalizable Robotic Manipulation](https://arxiv.org/abs/2605.28634)**：提出PrimitiveVLA，将机器人演示分解为可复用的运动基元，并使VLA模型学习这些不变的运动模式，从而大幅提升数据效率和零样本泛化能力。
*   **[Compositional Diffusion with Guided Search for Long-Horizon Planning](https://arxiv.org/abs/2601.00126)**：针对长时域任务规划中的“模式平均”问题，提出组合扩散与引导搜索方法。通过在扩散去噪过程中嵌入搜索，探索并组合局部生成模型的不同模态，生成全局一致且可行的长时域规划。
*   **[User-Driven Learning from Demonstration: A Trajectory and Impedance Learning Method](https://arxiv.org/abs/2607.16998)**：提出一种用户驱动的示教学习方法，通过融合3D快速微分同胚匹配算法和动态系统，实现单次示教的实时学习，既能高精度复现轨迹，又能学习阻抗变化以适应交互。
*   **[Leveraging Two Robotic Arms for Tight Assembly Performance Gains](https://arxiv.org/abs/2607.17876)**：提出一个用于双机械臂精密装配任务的端到端框架，证明双臂协调运动相比单臂可将平均执行时间缩短超过50%，并能生成更高质量的轨迹。

### 6. 其他机器人学与相关系统
*   **[A BIM-enabled, Agent-based Discrete-event Simulation Platform for Robotic Studies](https://arxiv.org/abs/2607.16920)**：提出一个基于建筑信息模型（BIM）的智能体仿真平台，利用图论为室内机器人提供环境信息和导航规划，支持在虚拟环境中评估复杂操作。
*   **[An Indoor Navigation System for the Visually Impaired based on UWB Positioning and D* Lite Path Planning Algorithm](https://arxiv.org/abs/2607.16614)**：为视障人士设计的室内导航系统，结合超宽带定位和D* Lite路径规划算法，在无GPS环境中提供精准定位和动态避障。
