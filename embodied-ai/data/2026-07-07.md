# Today's arXiv Papers Summary

## Embodied Agents & Foundation Models for Robotics

**[ACE-Brain-0.5: A Unified Embodied Foundational Model for Physical Agentic AI](https://arxiv.org/abs/2607.04426)**
A unified 8B model organizing robot intelligence into five coupled functions: spatial perception, decision making, embodied interaction, self-monitoring, and self-improvement. Introduces SSR+ for task-vector merging without cross-task interference. Improves over ACE-Brain-0 on 14/18 benchmarks.

**[InternVLA-A1.5: Unifying Understanding, Latent Foresight, and Action for Compositional Generalization](https://arxiv.org/abs/2607.04988)**
VLA policy using a native VLM backbone with a lightweight unified expert for continuous action. Uses learnable foresight tokens to condense future predictions, supervised by a frozen video generation model. Best overall results across 6 simulation benchmarks with preserved semantics.

**[WSA₁: A 3D-Centric World-Spatial-Action Model for Generalizable Robot Control](https://arxiv.org/abs/2607.03941)**
Novel RFM built on 3D world-aware modeling paradigm. Learns mutual constraints between 3D world state transitions and robotic actions. Achieves 93% success on RoboTwin2.0 with only 1k hours real robot data, +20% over SOTA RFMs on real tasks.

**[DSWAM: A Dual-System World Action Foundation Model for Fine-Grained Robot Manipulation](https://arxiv.org/abs/2607.04927)**
Dual-system with System 1 WAM executor and optional System 2 VLM subtask planner. Evaluated on DeMaVLA deformable manipulation with TensorRT acceleration and real-time chunking. Fair comparison against VLA policies.

**[Kairos: A Regret-Aware Native World-Action Model Stack for Physical AI](https://arxiv.org/abs/2606.16533)**
World-action model with cross-embodiment data curriculum, hybrid linear temporal attention, and deployment-aware system co-design. Achieves superior performance on embodied world-model and world-action benchmarks.

**[ABot-M0.5: Unified Mobility-and-Manipulation World Action Model](https://arxiv.org/abs/2607.00678)**
WAM with three-level alignment: temporal granularity via intermediate latent actions, action space via dual-level MoT, and train-test consistency via dream-forcing. SOTA on mobile manipulation benchmarks.

**[WAM4D: Fast 4D World Action Model via Spatial Register Tokens](https://arxiv.org/abs/2606.14048)**
Uses lightweight spatial register tokens as training-time future-depth readouts to transfer geometric priors into causal video-action transformer. Removes register branch for lightweight inference. Improves spatial consistency on RoboTwin 2.0.

## Vision-Language-Action (VLA) Policies

**[CAC-VLA: Context-Gated Action Conditioning for Vision-Language-Action Models](https://arxiv.org/abs/2607.04816)**
Learns latent-action interface within VLM. Predicts coarse-to-fine latent actions and adaptively leverages them via context gate. 98.3% average success on LIBERO, 89.5% on LIBERO-Plus.

**[CoRE-VLA: Towards Scalable and Robust Vision-Language-Action Modeling via Conditional Routing of Experts](https://arxiv.org/abs/2607.03693)**
Formulates action generation as context-conditioned sparse computation. Sensor availability gates modality-specialized experts for graceful degradation under missing sensors. Outperforms baselines on LIBERO and RoboCasa.

**[XS-VLA: Coupling Coarse-grained Spatial Distillation with Latent Flow Matching](https://arxiv.org/abs/2607.04171)**
Two-stage framework: distills spatial knowledge from Qwen3-VL-4B into SmolVLM2-0.25B, then uses latent flow matching policy with CVAE. SOTA among <0.5B models on LIBERO, 3.2x speedup over prior lightweight policy.

**[AnchorVLA: Bridging Discrete Decisions and Continuous Trajectories](https://arxiv.org/abs/2607.03182)**
Hierarchical decision-anchored VLA using trajectory-pattern anchors as explicit interface between reasoning and execution. SOTA 77.28 Success Rate on Bench2Drive closed-loop benchmark.

**[CamVLA: From Fixed to Free Cameras - Calibration-Free View-Robust VLA](https://arxiv.org/abs/2607.05396)**
Decouples manipulation from camera geometry by predicting camera-centric end-effector actions and hand-eye matrix. Calibration-free, depth-free, single-view, requiring only monocular RGB. Improves success across unseen viewpoints.

**[From Mobile Data to Business Insights... Wait, wrong paper. Here: TACO](https://arxiv.org/abs/2607.02840)**
Tactile-aware world model for VLA post-training in contact-rich tasks. Uses Recognize-Imagine-Label loop with visuo-tactile generation. 44% absolute success rate improvement over base policy.

**[SEAM: Smooth Execution of Action-Chunked Motion for VLA Policies](https://arxiv.org/abs/2607.04609)**
Training-free inference-time method for flow matching VLAs. Uses velocity-guided loss steering from previous chunk's unexecuted tail. Reduces boundary jerk by 28%, chunk transition discontinuity by 27%.

**[VLA Grounder: Language-Conditioning Space Optimization for Black-Box VLA](https://arxiv.org/abs/2607.04517)**
Optimizes language space rather than action weights for frozen VLA policies. RL discovers VLA-grounded commands that best elicit successful behavior. Improves on instruction-sensitive tasks.

**[BiliVLA: Scene-Aware VLA with RL for Autonomous Biliary Endoscopic Navigation](https://arxiv.org/abs/2606.23531)**
VLA for endoscopic navigation combining grounding-enhanced SFT with GRPO. 84.85% success rate on ERCP subtasks with 91.96% action precision.

**[Exp2VLA: Enabling VLA for Drone Navigation from Expert Demonstrations](https://arxiv.org/abs/2607.03146)**
Distills expert behavior from RL/teleoperation into training data for compact VLA models for drone navigation. Handles varied semantic commands and generalizes to unseen target compositions.

## Tactile Sensing & Contact-Rich Manipulation

**[ResTacVLA for Contact-Rich Manipulation via Residual Tactile Representation](https://arxiv.org/abs/2607.03387)**
Reformulates tactile data as residual representation capturing discrepancy between visual priors and physical sensations. Uses VQ bottleneck into Latent Contact Primitives. Outperforms baselines on contact-rich tasks with robustness to disturbances.

**[OmniTacTune: Policy-Agnostic Real-World RL for Tactile Residual Adaptation](https://arxiv.org/abs/2607.03723)**
Two-stage pipeline: bootstraps tactile-aware learning from autonomous rollouts, then learns lightweight tactile residual policy. Improves visual base policies from 5-40% to 85-100% in 40-80 minutes.

**[CoorGrasp: Coordinated Contact Control for Adaptive Dexterous Grasping](https://arxiv.org/abs/2607.03557)**
Tactile-driven MPC for adaptive dexterous grasp execution. Features coordination-aware phase separation, arm-hand coordination, and adaptive force coordination. Validated on 15k grasps across 478 objects.

**[Current as Touch: Proprioceptive Contact Feedback for Compliant Dexterous Manipulation](https://arxiv.org/abs/2607.03529)**
Learns contact-aware compliance cues from motor current and joint states without external tactile sensors. Predicts compliance reference position for PD controller. Enables fragile object handling, sustained contact, and thin-object retrieval.

**[Metallic Ultrasound Waveguides as a Distributed Tactile Sensing Platform](https://arxiv.org/abs/2607.02827)**
Investigates metallic waveguides as distributed tactile sensors with single transducer. Linear force estimation (R²≥0.95), material classification, contact localization. Extended to 2D sheets.

**[GelNeuro: A Sensing-Computing Integrated Neuromorphic Tactile System](https://arxiv.org/abs/2607.05241)**
Pairs GelSight Mini tactile sensor with Speck2f neuromorphic SoC. Spiking CNN classifier achieves 96.3% accuracy on 15-class texture recognition at only 19.6 mW.

## Dexterous Manipulation & Grasping

**[HUGS: Guiding Unified Dexterous Grasp Synthesis Across Modes and Scales via Learned Human Priors](https://arxiv.org/abs/2607.04554)**
Learns object-conditioned human prior grasping preferences and guides force-closure optimization. Synthesizes 3.2M grasps spanning 2cm-30cm objects, from two-finger to bimanual grasps.

**[Cross-Embodiment Robot Manipulation via a Unified Hand Action Space](https://arxiv.org/abs/2607.03570)**
Sphere-based unified action representation (UHAS) for cross-embodiment dexterous manipulation. Uses Cascade Inverse Kinematics to map to embodiment-specific joints. Zero-shot transfer across Allegro, LEAP, Shadow, and MANO hands.

**[Closing the Reality Gap: Zero-Shot Sim-to-Real for Dexterous Force-Based Grasping](https://arxiv.org/abs/2607.04940)**
Sim-to-real RL for five-finger hand with fast tactile simulation, current-to-torque calibration, and actuator dynamics randomization. Trained entirely in simulation, zero-shot transfer for controllable grasp force and in-hand reorientation.

**[SILO: Simulation-in-the-Loop Sim-to-Real for Multi-Stage Cable Routing](https://arxiv.org/abs/2607.04616)**
Sim-to-real RL for cable routing using GPU-parallelized simulation. Novel SILO deployment framework combining localized RL policies and cable state estimation. 2x reduction in cycle times.

## Imitation Learning & Policy Learning

**[Continuous Latent Action Flow Policy (CoLA-Flow Policy)](https://arxiv.org/abs/2601.23087)**
Flow matching in continuous latent action space. Encodes action sequences into temporally coherent latent trajectories. Up to 93.7% smoother trajectories, 25pp improvement over raw action-space flow baselines.

**[Geometry-Aware Motion Latents for Learning Robust Manipulation Policies](https://arxiv.org/abs/2607.04714)**
Learns discrete motion latent codes by predicting point cloud evolution during manipulation. SOTA performance using single-view RGB-D input. Physical transformations regardless of visual context.

**[ChronoFlow-Policy: Unifying Past-Current-Future Interaction Flow](https://arxiv.org/abs/2606.31493)**
Temporally unified representation capturing past, current, and future interaction dynamics through sparse 3D keypoints. Outperforms diffusion policy baselines on long-horizon manipulation.

**[Guided Action Flow: Q-Guided Inference for Flow-Matching VLA Policies](https://arxiv.org/abs/2607.02092)**
Inference-time framework keeping pretrained VLA frozen, uses learned action-chunk critic to guide reverse-time flow sampler. Improves LIBERO success from 68% to 82%.

**[Simple-to-Complex Structured Demonstrations for VLA Learning](https://arxiv.org/abs/2607.04591)**
Proposes demonstration organization strategy decomposing tasks into progressively learnable sub-skills. Improves success rate and training stability on block grasping and towel folding.

**[Beyond Monotonic Progress: Retry-Supervised Value Learning](https://arxiv.org/abs/2606.24633)**
Learns mistake-sensitive value functions from mixed-quality demonstrations using retry events as sparse supervision. Improves imitation learning from imperfect demonstrations on real tasks.

**[AutoSpeed: Annotation-Free Stage-Adaptive Motion Speed Learning](https://arxiv.org/abs/2607.01051)**
Enables stage-adaptive motion speeds via DCT-based frequency domain modulation. Reduces execution time while improving success rates.
