Here is a curated summary of today's arXiv papers, focusing on embodied intelligence, manipulation, and related topics.

### Dexterous Manipulation & Visuomotor Policies

*   **[TransDex: Pre-training Visuo-Tactile Policy with Point Cloud Reconstruction for Dexterous Manipulation of Transparent Objects](https://arxiv.org/abs/2603.13869)**: Proposes a self-supervised pre-training method for dexterous manipulation of transparent objects, which are difficult for standard depth sensors. The model uses point cloud reconstruction to learn 3D structure, then fuses vision and tactile data for a manipulation policy.

*   **[AnchorRefine: Synergy-Manipulation Based on Trajectory Anchor and Residual Refinement for Vision-Language-Action Models](https://arxiv.org/abs/2604.17787)**: Introduces a hierarchical framework for VLA policies that factorizes action generation into coarse trajectory anchors and residual refinements. This allows the model to learn global movement and local corrective actions separately, improving performance on precision-critical tasks.

*   **[ImplicitRDP: An End-to-End Visual-Force Diffusion Policy with Structural Slow-Fast Learning](https://arxiv.org/abs/2512.10946)**: A unified policy for contact-rich manipulation that integrates visual planning and reactive force control. It uses a "structural slow-fast learning" mechanism to handle the different frequencies of vision (slow) and force (fast) signals.

### Teleoperation, Data Collection & Skill Acquisition

*   **[ModPack: An Extensible Teleoperation Interface for Bimanual Mobile Manipulation](https://arxiv.org/abs/2607.19479)**: Presents a modular teleoperation system built around a wearable "backpack" with onboard compute. It supports plug-and-play modules for tasks like joint-level teleoperation and mobile manipulation, designed for scalable data collection.

*   **[Robots Acquire Manipulation Skills in Seconds from a Single Human Video](https://arxiv.org/abs/2607.20033)**: Introduces HOST, a framework enabling a robot to learn a new skill from a single human video in about 29 seconds. It works by mapping both the robot trajectory and the video to a shared task-progress manifold and predicting future actions.

*   **[EgoRecovery: Acquiring Failure Recovery Ability Through Human Recovery Demonstration](https://arxiv.org/abs/2607.19745)**: Proposes using egocentric human data of failure recovery to teach robots how to recover from their own failures. Human demonstrations are aligned to a shared "corrective-intent" space, requiring only a small amount of robot data to connect intent to action.

### Tactile & Force Sensing in Manipulation

*   **[V2F: Vision-Informed Grasp Force Prediction for Damage-Aware Robotic Handling of Date Fruits](https://arxiv.org/abs/2607.19804)**: Develops a vision-to-force pipeline that predicts a safe grasp force for delicate objects like date fruits. It uses visual features to estimate forces needed to secure objects without causing damage, reducing the need for slow tactile exploration.

### Sim-to-Real & World Models for Manipulation

*   **[KineBench: Benchmarking Embodied World Models via IDM-Free Kinematic Grounding](https://arxiv.org/abs/2607.19876)**: A new benchmark for evaluating the physical consistency of embodied world models. It avoids using learned Inverse Dynamics Models for action extraction, instead directly extracting 3D end-effector poses from generated video frames for more reliable closed-loop evaluation.

### Robots & Hardware for Embodied AI

*   **[Milo, a Fully Autonomous Indoor/Outdoor Robotic Guide Dog](https://arxiv.org/abs/2607.19530)**: Presents an open-source, low-cost robotic guide dog platform. It is fully autonomous with onboard computation and a perception stack for indoor and outdoor navigation with obstacle avoidance.

*   **[The Open Ant: A Robot Platform for Reinforcement Learning Research](https://arxiv.org/abs/2607.18488)**: A physical robot platform designed to be a low-cost, robust variant of the common Gymnasium Ant environment. The goal is to make it easy for RL researchers to perform physical robot experiments.

### Navigation & Mobile Manipulation

*   **[NavVerse: Benchmarking Indoor-to-Outdoor Embodied Navigation in Continuous Robot Simulation](https://arxiv.org/abs/2607.19695)**: A physics-enabled benchmark for embodied navigation that requires agents to navigate from indoor to outdoor scenes in a single episode, evaluating them on task success, path efficiency, and safety.

*   **[EA-Nav: Learning Safe Visual Navigation Policies with Embodiment Awareness](https://arxiv.org/abs/2607.19880)**: An imitation-learning framework for cross-embodiment navigation that incorporates the robot's geometry as a conditional token to resolve action ambiguity when different robots see the same scene.

### Manipulation Planning & Grasping

*   **[LENS: LLM-guided Environment Simplification for Planning and Control in Clutter](https://arxiv.org/abs/2607.19633)**: Proposes using LLMs to automatically simplify cluttered scenes for robotic manipulation by merging or pruning objects. This plug-and-play fix improves the performance of various downstream planners and VLA models.

*   **[SeededGrasp: Language-Guided Grasping in Complex Scenes with Multiple Embodiments](https://arxiv.org/abs/2607.20207)**: A data-efficient framework that uses a VLM to predict a "seed point" on an object, which then conditions a lightweight grasp generation model. This decoupling enables multi-embodiment support without expensive end-to-end training.

### Simulation & Policy Learning

*   **[Diffusion ReRoll: Revisable Denoising for Robotic Sequential Prediction](https://arxiv.org/abs/2607.19919)**: A novel diffusion framework for sequential prediction that allows for iterative revision across the prediction horizon. It selectively re-noises and refines parts of the sequence, improving performance on planning and policy learning tasks.

*   **[Zero2Skill: Bootstrapping Robot Skills through Autonomous Data Collection, Training, and Deployment](https://arxiv.org/abs/2607.14047)**: A human-robot symbiotic system for data collection where corrections from a remote operator are stored in a "Corrective Memory" and reused. This reduces human oversight as known failure modes do not need to be corrected again.

### Modeling & Control

*   **[MorphQuad: Morphable Quadrotor for Superhuman Maneuverability, Manipulation, and Resiliency](https://arxiv.org/abs/2607.02764)**: A morphable quadrotor with independently articulated rotors, designed for extreme maneuverability and contact-based manipulation. It demonstrates tasks like valve turning, perching, and object pushing.

*   **[Contact-Persistent Full Actuation for Aerial Physical Interaction](https://arxiv.org/abs/2607.19708)**: Introduces a framework to certify that a UAV can remain fully actuated during sustained contact with an environment. It defines residual authority margins to ensure the robot can stabilize and reject disturbances while interacting.
