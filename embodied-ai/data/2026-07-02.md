### Dexterous Manipulation & Multi-finger Grasping

- [CHORD (Paper 11)](https://arxiv.org/abs/2607.00033): A framework for learning dexterous manipulation policies by using object-centric contact wrench guidance from human demonstrations. It scales to a large benchmark of 4,739 bimanual tasks and transfers to real-world robots.
- [H-Tac & TTP (Paper 45)](https://arxiv.org/abs/2607.01067): Proposes a large-scale tactile-action dataset from egocentric human videos (H-Tac) and a Transferable Tactile Pre-training (TTP) system to bridge the human-robot gap for dexterous manipulation.
- [Labimus (Paper 81)](https://arxiv.org/abs/2606.31037): Introduces the first benchmark for humanoid dexterous manipulation in organic chemistry labs, featuring functional lab instruments and powder physics.
- [KungfuBot (Paper 64)](https://arxiv.org/abs/2506.12851): A physics-based framework for humanoid whole-body control to master highly-dynamic skills like Kungfu by processing and tracking human motions with adaptive tolerance.
- [Multi-Embodiment Retargeting (Paper 63)](https://arxiv.org/abs/2505.20857): A diffusion-based framework for retargeting motion patterns across diverse robot embodiments using graph-structured representations.

### Visuomotor Policies & Imitation Learning

- [VolumeDP (Paper 68)](https://arxiv.org/abs/2603.17720): Proposes a policy architecture that lifts 2D image features into a 3D volumetric representation (VolumeDP) for explicit spatial reasoning, achieving state-of-the-art results on LIBERO.
- [3DPWM (Paper 17)](https://arxiv.org/abs/2607.00148): A 3D world model that completes partial point clouds before learning action-conditioned dynamics, enabling reliable long-horizon rollouts for model-based planning.
- [DICE-RL (Paper 67)](https://arxiv.org/abs/2603.10263): A framework that uses RL as a "distribution contraction" operator to refine pretrained generative robot policies, turning a behavior prior into a high-performing "pro" policy.
- [AutoSpeed (Paper 43)](https://arxiv.org/abs/2607.01051): A model-agnostic framework enabling visuomotor policies to predict trajectories with stage-adaptive motion speeds without requiring speed annotations.
- [Language-Critique Imitation Learning (Paper 262)](https://arxiv.org/abs/2607.01225): Uses natural language as structured supervision to learn from suboptimal demonstrations, avoiding the collapse of feedback into scalar signals.

- [RoboWorld (Paper 44)](https://arxiv.org/abs/2607.01060): An automated evaluation pipeline with a fast autoregressive video world model for evaluating generalist robot policies, achieving high correlation with real-world evaluation.
- [Structured 4D Latent Predictive Model (Paper 50)](https://arxiv.org/abs/2607.01166): Predicts the evolution of a scene's 3D structure in a structured latent space for robot planning, demonstrating strong 3D consistency and superior performance on manipulation tasks.
- [Warp RL (Paper 94)](https://arxiv.org/abs/2606.31043): Replaces additive residuals with an invertible state-conditioned transformation of a policy's action distribution for dynamics adaptation, showing 30% faster task completion on a real-robot peg-insertion task.
- [ABot-M0.5 (Paper 57)](https://arxiv.org/abs/2607.00678): A World Action Model (WAM) designed for mobile manipulation that aligns temporal granularity, action space, and train-test consistency for long-horizon tasks.

- [Joint Discovery of Object/Action Symbols (Paper 10)](https://arxiv.org/abs/2607.00031): Proposes a model that jointly discovers manipulation primitives and object categories through effect prediction, improving planning precision on seen and novel objects.
