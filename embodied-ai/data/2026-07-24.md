### Dexterous Manipulation, Grasping, and In-Hand Manipulation

*   **[Grasp, Handover, Rotate: Bimanual Object Reorientation via Compositional Diffusion and Energy-Based Optimization](https://arxiv.org/abs/2607.21341)**
    *   **BiCompoDiff**: A framework for bimanual object reorientation that combines grasp diffusion with energy-based planning to handle collisions, kinematic constraints, and handover. Achieves 20% higher success rates and smoother trajectories than baselines.

*   **[FORGE-plus: Force-Budgeted Recovery for Contact-Rich Assembly with a Frozen LLM Supervisor](https://arxiv.org/abs/2607.21227)**
    *   A two-layer framework for tight-clearance assembly. A frozen LLM sets a force ceiling and selects recovery maneuvers, while a low-level RL policy executes within that budget. Achieves 256/256 successful fragile-object placements without breakage.

### Visuomotor Policies and Imitation Learning

*   **[GuidedAttention: Interpretable and Correctable Visual Attention for OOD-Robust Robot Manipulation via Imitation Learning](https://arxiv.org/abs/2607.21049)**
    *   Introduces interpretable visual attention keypoints as an intermediate representation in a diffusion policy. Users can inspect and correct attention at rollout, improving performance under out-of-distribution conditions like novel positions and appearances.

*   **[FELT: Generating Tactile Signals from Vision for Visuo-Tactile Manipulation](https://arxiv.org/abs/2607.20683)**
    *   A framework that synthesizes tactile pressure images from RGB observations using a frozen visual encoder. This augments vision-only data with tactile information, improving policy success in contact-rich tasks like insertion and in-hand rotation without needing physical tactile sensors at test time.

*   **[URF: A Unified Robot Control-Policy Framework for Stable Contact Aware Manipulation](https://arxiv.org/abs/2607.20912)**
    *   Proposes predicting not only a compliant action (virtual target) but also the controller's behavior (stiffness and impedance-admittance switch ratio). This unified approach improves task success and reduces failure modes like excessive force and tool breakage during contact-rich tasks.

*   **[Force-Aware Residual DAgger via Trajectory Editing for Precision Insertion with Impedance Control](https://arxiv.org/abs/2603.04038)**
    *   **TER-DAgger**: A human-in-the-loop IL framework that mitigates covariate shift. It uses trajectory editing to fuse policy rollouts with human corrections and a force-aware failure anticipation mechanism to trigger human intervention only when needed, improving insertion success rates by over 37%.

*   **[VPWEM: Non-Markovian Visuomotor Policy with Working and Episodic Memory](https://arxiv.org/abs/2603.04910)**
    *   A diffusion policy that adds a Transformer-based memory compressor to store long-term episodic memory. Outperforms SOTA baselines on memory-intensive manipulation tasks (20%+ improvement) while maintaining near-constant computation per step.

### Tactile Sensing and Sim-to-Real

*   **[PhysCoRe: Physics-Corrected Residual World Models for Material-Aware Deformable Dynamics](https://arxiv.org/abs/2607.20653)**
    *   A world model that couples a differentiable material point method (MPM) simulator with neural networks. It infers per-particle elasticity from visual observations and learns residual corrections for simulation biases, enabling accurate prediction of deformable object dynamics and online material identification.

### Teleoperation and Data Collection

*   **[AXIS: A Growable Community-Driven Data Engine for Scalable Robot Manipulation](https://arxiv.org/abs/2607.21588)**
    *   A platform for community-driven data collection with browser-based teleoperation. Provides a dataset of 207 tasks and 50K+ trajectories. Continual pretraining with AXIS data substantially improves the success rate of a VLA policy (\(\pi_{0.5}\)) by 5.8%.

*   **[Robots Acquire Manipulation Skills in Seconds from a Single Human Video](https://arxiv.org/abs/2607.20033)**
    *   **HOST**: A framework for one-shot skill acquisition from a single human video. It uses a self-grounded cascade: estimating progress, predicting future observations, and deriving actions. Acquires novel skills in ~29 seconds and outperforms zero-shot baselines by 45%.

*   **[TableVerse: A Large-scale Tabletop Dataset with Real-world Grounded Layouts for Generalizable Manipulation](https://arxiv.org/abs/2607.21017)**
    *   A Real2Sim pipeline that reconstructs high-fidelity, physically plausible tabletop environments from in-the-wild images. The resulting TableVerse-100K dataset contains 100,000 unique scenes with pick-and-place demonstrations to facilitate generalizable manipulation research.

### Embodied AI, Navigation & Task Planning

*   **[Robostral Navigate](https://arxiv.org/abs/2607.20785)**
    *   An 8B VLM for navigation that predicts waypoints from a stream of monocular RGB images. It sets a new state-of-the-art on R2R-CE (77.4% success) using only a single camera, surpassing methods that rely on depth sensors or multi-camera rigs.

*   **[ZONDA: Zero-shot Object Navigation with Dynamic Avoidance in Multi-floor Environments](https://arxiv.org/abs/2607.21025)**
    *   A zero-shot navigation framework for multi-floor environments that handles dynamic pedestrians. It uses heuristic multi-floor planning and multi-view target verification with a VLM to reduce false positives.

*   **[Emergent Compositional Skills in Mixture-of-Experts VLAs](https://arxiv.org/abs/2607.20771)**
    *   An analysis of VLAs trained with a Mixture-of-Experts (MoE) action head. Shows that the model's router learns to compose tasks by activating distinct experts, which correspond to reusable, interpretable low-level primitives, offering a path towards modular and interpretable robot policies.

### World Models & Generalization

*   **[Do World Action Models Generalize Better than VLAs? A Robustness Study](https://arxiv.org/abs/2603.22078)**
    *   A comparative study of VLAs and World Action Models (WAMs) under visual and language perturbations. WAMs demonstrate stronger robustness in many scenarios, while VLAs require extensive training to be comparable, suggesting the importance of video-based dynamic priors.

*   **[PhysCoRe: Physics-Corrected Residual World Models for Material-Aware Deformable Dynamics](https://arxiv.org/abs/2607.20653)**
    *   *(Also listed under Tactile Sensing)* A residual world model for deformable objects that combines a differentiable physics simulator with learned residual corrections, enabling accurate prediction and material identification.

### Scaling Data & Evaluation

*   **[Scale Up Strategically: Learning Compositional Generalization via Bias-Aware Evaluation and Data Collection](https://arxiv.org/abs/2607.21582)**
    *   A framework to diagnose and quantify "instruction factor bias" (e.g., over-reliance on color vs. object name). A bias-aware data collection strategy that reallocates budget towards under-grounded factors (e.g., verb, size) improves sample efficiency and generalization.

*   **[TOPrewward: Token Probabilities as Hidden Zero-Shot Rewards for Robotics](https://arxiv.org/abs/2602.19313)**
    *   A training-free reward method that uses the internal token probabilities from a VLM to provide dense, instruction-conditioned progress rewards. This eliminates the need for task-specific reward model training or manual labeling.

*   **[What Matters for Simulation to Online Reinforcement Learning on Real Robots](https://arxiv.org/abs/2602.20220)**
    *   A large-scale empirical study (100 real-world training runs) that identifies robust algorithmic, systems, and experimental design choices for successful online RL on physical robots, enabling practitioners with lower engineering effort.
